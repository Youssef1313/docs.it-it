## <a name="microservices"></a><span data-ttu-id="628f0-135">Microservizi</span><span class="sxs-lookup"><span data-stu-id="628f0-135">Microservices</span></span>

- <span data-ttu-id="628f0-140">La gestione degli errori diventa più efficiente perché i problemi possono essere tracciati in servizi specifici che vengono quindi isolati, ricompilati, testati e ridistribuiti indipendentemente dagli altri servizi.</span><span class="sxs-lookup"><span data-stu-id="628f0-140">Error handling becomes more efficient as problems can be traced to specific services that are then isolated, rebuilt, tested, and redeployed independent of the other services.</span></span>
