# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernizza le app .NET esistenti in applicazioni ottimizzate per il cloud

> Obiettivo: Modernizza le tue applicazioni di .NET Framework esistenti con applicazioni ottimizzate per il cloud per migliorare drasticamente la flessibilità della distribuzione, in modo da ottenere i costi di distribuzione delle applicazioni più veloci e inferiori.

Per sfruttare i vantaggi del cloud e le nuove tecnologie come i contenitori, è consigliabile modernizzare almeno parzialmente le applicazioni .NET esistenti. In definitiva, la modernizzazione delle applicazioni aziendali consentirà una riduzione del costo totale di proprietà.

La modernizzazione parziale di un'app non implica necessariamente una migrazione e una riarchitettura complete. È possibile modernizzare inizialmente le applicazioni esistenti con una modernizzazione importante ma facile da realizzare. È possibile mantenere il codice corrente, scritto in versioni di .NET Framework esistenti, con tutte le dipendenze di Windows e IIS. La figura 4-1 illustra in che modo le app ottimizzate per il cloud sono posizionate in modelli maturi per la modernizzazione delle applicazioni.
