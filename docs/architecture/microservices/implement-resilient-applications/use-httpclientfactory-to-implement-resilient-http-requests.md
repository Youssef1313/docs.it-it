### <a name="httpclient-lifetimes"></a><span data-ttu-id="46b9b-148">Durate di HttpClient</span><span class="sxs-lookup"><span data-stu-id="46b9b-148">HttpClient lifetimes</span></span>

<span data-ttu-id="46b9b-151">Il pooling dei gestori è opportuno poiché ogni gestore si occupa in genere di gestire le proprie connessioni HTTP sottostanti.
