<span data-ttu-id="cf84a-103">Un *campo* è una variabile di qualsiasi tipo che viene dichiarata direttamente in una [classe](../../language-reference/keywords/class.md) o [struct](../../language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="cf84a-103">A *field* is a variable of any type that is declared directly in a [class](../../language-reference/keywords/class.md) or [struct](../../language-reference/keywords/struct.md).</span></span> <span data-ttu-id="cf84a-104">I campi sono *membri* del rispettivo tipo contenitore.</span><span class="sxs-lookup"><span data-stu-id="cf84a-104">Fields are *members* of their containing type.</span></span>
