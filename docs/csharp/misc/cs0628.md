# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="d8a86-102">Avviso del compilatore (livello 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="d8a86-102">Compiler Warning (level 4) CS0628</span></span>

 <span data-ttu-id="d8a86-104">Una classe [sealed](../language-reference/keywords/sealed.md) non può introdurre un membro [protected](../language-reference/keywords/protected.md) perché nessun'altra classe potrà ereditare dalla classe `sealed` e usare il membro `protected` .</span><span class="sxs-lookup"><span data-stu-id="d8a86-104">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="d8a86-105">L'esempio seguente genera l'errore CS0628:</span><span class="sxs-lookup"><span data-stu-id="d8a86-105">The following sample generates CS0628:</span></span>

```csharp  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
