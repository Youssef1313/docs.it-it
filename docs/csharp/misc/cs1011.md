# <a name="compiler-error-cs1011"></a>Errore del compilatore CS1011
Il valore letterale carattere è vuoto  
  
 [char](../language-reference/keywords/char.md) è stato dichiarato e inizializzato in un valore Null. L'inizializzazione di `char` deve specificare un carattere.  
  
 L'esempio seguente genera l'errore CS1011:  
  
```csharp  
// CS1011.cs  
class Sample  
{  
   public char CharField = '';   // CS1011  
}  
```
