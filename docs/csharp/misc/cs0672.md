<span data-ttu-id="7dce2-103">Il membro 'member1' esegue l'override del membro obsoleto 'member2'.</span><span class="sxs-lookup"><span data-stu-id="7dce2-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="7dce2-104">Aggiungere l'attributo Obsolete a 'member1'</span><span class="sxs-lookup"><span data-stu-id="7dce2-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="7dce2-105">Il compilatore ha trovato un `override` di un metodo contrassegnato come `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="7dce2-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="7dce2-106">Il metodo che esegue l'override, però, non è contrassegnato come obsoleto.</span><span class="sxs-lookup"><span data-stu-id="7dce2-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="7dce2-107">Se viene chiamato il metodo che esegue l'override, verrà generato comunque l'errore [CS0612](./cs0612.md).</span><span class="sxs-lookup"><span data-stu-id="7dce2-107">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="7dce2-108">Esaminare le dichiarazioni del metodo e indicare in modo esplicito se un metodo (e tutti i relativi override) deve essere contrassegnato come `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="7dce2-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="7dce2-109">L'esempio seguente genera l'errore CS0672:</span><span class="sxs-lookup"><span data-stu-id="7dce2-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
