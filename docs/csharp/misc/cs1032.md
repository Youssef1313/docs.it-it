 <span data-ttu-id="f81a2-105">L'esempio seguente genera l'errore CS1032:</span><span class="sxs-lookup"><span data-stu-id="f81a2-105">The following sample generates CS1032:</span></span>

```csharp  
{  
   public class clx  
   {  
      #define a   // CS1032, put before namespace

      }  
   }  
}  
```
