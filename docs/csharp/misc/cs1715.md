# <a name="compiler-error-cs1715"></a><span data-ttu-id="0bd8f-102">Errore del compilatore CS1715</span><span class="sxs-lookup"><span data-stu-id="0bd8f-102">Compiler Error CS1715</span></span>
<span data-ttu-id="0bd8f-103">'Type1': il tipo deve essere 'Type2' in modo che corrisponda al membro 'MemberName' sottoposto a override</span><span class="sxs-lookup"><span data-stu-id="0bd8f-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="0bd8f-104">Questo errore è identico [Compiler Error CS0508](./cs0508.md), ad eccezione del fatto che CS0508 ora si applica solo ai metodi che hanno tipi restituiti, mentre CS1715 si applica alle proprietà e agli indicizzatori che hanno solo 'tipi' invece di 'tipi restituiti'.</span><span class="sxs-lookup"><span data-stu-id="0bd8f-104">This error is the same as [Compiler Error CS0508](./cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0bd8f-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="0bd8f-105">Example</span></span>  
 <span data-ttu-id="0bd8f-106">Il codice seguente genera l'errore CS1715.</span><span class="sxs-lookup"><span data-stu-id="0bd8f-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
