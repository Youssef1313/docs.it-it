# <a name="compiler-error-cs0238"></a><span data-ttu-id="75963-102">Errore del compilatore CS0238</span><span class="sxs-lookup"><span data-stu-id="75963-102">Compiler Error CS0238</span></span>
<span data-ttu-id="75963-103">'member' non può essere sealed perché non è un override</span><span class="sxs-lookup"><span data-stu-id="75963-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="75963-104">[sealed](../language-reference/keywords/sealed.md) è stato usato in un membro non contrassegnato anche con [override](../language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="75963-104">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="75963-105">Per altre informazioni, vedere [Ereditarietà](../programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="75963-105">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="75963-106">L'esempio seguente genera l'errore CS0238:</span><span class="sxs-lookup"><span data-stu-id="75963-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
