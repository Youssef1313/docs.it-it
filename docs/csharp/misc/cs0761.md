# <a name="compiler-error-cs0761"></a><span data-ttu-id="c501e-102">Errore del compilatore CS0761</span><span class="sxs-lookup"><span data-stu-id="c501e-102">Compiler Error CS0761</span></span>
<span data-ttu-id="c501e-103">Le dichiarazioni di metodo parziale di '\<Method T >' presentano vincoli di parametro di tipo incoerenti.</span><span class="sxs-lookup"><span data-stu-id="c501e-103">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="c501e-104">Se un metodo parziale ha un'implementazione, il vincolo di tipo generico deve essere identico al vincolo definito nella firma del metodo.</span><span class="sxs-lookup"><span data-stu-id="c501e-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c501e-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="c501e-105">To correct this error</span></span>  
  
1. <span data-ttu-id="c501e-106">Rendere i vincoli di tipo generico identici in ogni parte del metodo parziale.</span><span class="sxs-lookup"><span data-stu-id="c501e-106">Make the generic type constraints identical on each part of the partial method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c501e-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="c501e-107">Example</span></span>  
 <span data-ttu-id="c501e-108">Il codice seguente genera l'errore CS0761:</span><span class="sxs-lookup"><span data-stu-id="c501e-108">The following code generates CS0761:</span></span>

```csharp  
{  
    partial void Part<T>() where T : class;

    }  
}  
```
