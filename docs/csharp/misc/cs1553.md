# <a name="compiler-error-cs1553"></a>Errore del compilatore CS1553

Il tipo restituito per un [operatore di conversione](../../csharp/language-reference/operators/user-defined-conversion-operators.md) deve precedere l'elenco dei parametri e *modificatore* sia `implicit` o `explicit`.
  
L'esempio seguente genera l'errore CS1553:  
  
```csharp  
// CS1553.cs  
class MyClass  
{  
   public static int implicit operator (MyClass f)   // CS1553  
   // try the following line instead  
   // public static implicit operator int (MyClass f)  
   {  
      return 6;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
