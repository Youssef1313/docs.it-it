## <a name="example"></a><span data-ttu-id="be164-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="be164-106">Example</span></span>  
 <span data-ttu-id="be164-107">L'esempio seguente genera l'errore CS0540.</span><span class="sxs-lookup"><span data-stu-id="be164-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
