# <a name="compiler-error-cs1555"></a>Errore del compilatore CS1555
Impossibile trovare 'class' specificata per il metodo Main  
  
 Nell'opzione del compilatore [/main](../language-reference/compiler-options/main-compiler-option.md) è stata specificata una classe, ma il nome della classe non è stato trovato nel codice sorgente.
