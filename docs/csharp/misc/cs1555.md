# <a name="compiler-error-cs1555"></a>Errore del compilatore CS1555
Impossibile trovare 'class' specificata per il metodo Main  
  
 È stata specificata una classe per l'opzione del compilatore [-Main](../language-reference/compiler-options/main-compiler-option.md) , ma il nome della classe non è stato trovato nel codice sorgente.
