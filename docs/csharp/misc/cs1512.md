 <span data-ttu-id="0c86d-104">La parola chiave [base](../language-reference/keywords/base.md) è stata usata al di fuori di un metodo, di una proprietà o di un costruttore.</span><span class="sxs-lookup"><span data-stu-id="0c86d-104">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>

```csharp

class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
