# <a name="compiler-error-cs0056"></a>Errore del compilatore CS0056
Accessibilità incoerente: il tipo restituito 'type' è meno accessibile dell'operatore 'operator'  
  
 Un costrutto pubblico deve restituire un oggetto accessibile pubblicamente. Per altre informazioni, vedere [Access Modifiers](../programming-guide/classes-and-structs/access-modifiers.md) (Modificatori di accesso).  
  
 L'esempio seguente genera l'errore CS0056:  
  
```csharp  
// CS0056.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class A  
{  
   public static implicit operator MyClass(A a)   // CS0056  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
