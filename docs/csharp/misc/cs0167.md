# <a name="compiler-error-cs0167"></a>Errore del compilatore CS0167
Nel delegato 'delegate' manca il metodo Invoke  
  
 È stato importato e usato un programma (che usa Common Language Runtime di .NET Framework) creato con un altro compilatore. Il compilatore ha consentito un oggetto [delegate](../language-reference/builtin-types/reference-types.md)con un formato non corretto. Pertanto, il metodo `Invoke` non è disponibile. Per altre informazioni, vedere [Delegati](../programming-guide/delegates/index.md).
