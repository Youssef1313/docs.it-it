# <a name="compiler-warning-level-1-cs2029"></a>Avviso del compilatore (livello 1) CS2029
Valore non valido per '/define'. 'identifier' non è un identificatore valido  
  
 Questo avviso viene visualizzato se il valore usato per l'opzione [/define](../language-reference/compiler-options/define-compiler-option.md) contiene caratteri non validi.  
  
 Questo avviso non può essere eliminato con l'opzione [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
