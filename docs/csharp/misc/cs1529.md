## <a name="example"></a><span data-ttu-id="51c61-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="51c61-105">Example</span></span>  
 <span data-ttu-id="51c61-106">L'esempio seguente genera l'errore CS1529:</span><span class="sxs-lookup"><span data-stu-id="51c61-106">The following sample generates CS1529:</span></span>

```csharp  
    {  
        using Microsoft;

```
