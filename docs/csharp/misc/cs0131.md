 <span data-ttu-id="50058-104">In un'istruzione di assegnazione il valore della parte destra viene assegnato alla parte sinistra.</span><span class="sxs-lookup"><span data-stu-id="50058-104">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="50058-105">La parte sinistra deve essere una variabile, una proprietà o un indicizzatore.</span><span class="sxs-lookup"><span data-stu-id="50058-105">The left-hand side must be a variable, property, or indexer.</span></span>

## <a name="example"></a><span data-ttu-id="50058-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="50058-108">Example</span></span>  
 <span data-ttu-id="50058-109">L'esempio seguente genera l'errore CS0131.</span><span class="sxs-lookup"><span data-stu-id="50058-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="50058-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="50058-110">Example</span></span>  
 <span data-ttu-id="50058-111">Questo errore può verificarsi anche se si tenta di eseguire operazioni aritmetiche nella parte sinistra dell'operatore di assegnazione, come nell'esempio seguente.</span><span class="sxs-lookup"><span data-stu-id="50058-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
