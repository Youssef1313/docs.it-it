# <a name="compiler-error-cs2016"></a>Errore del compilatore CS2016
La tabella codici 'codepage' non è valida o non è installata  
  
 All'opzione del compilatore [/codepage](../language-reference/compiler-options/codepage-compiler-option.md) è stato passato un valore non valido.  
  
 L'esempio seguente genera l'errore CS2016:  
  
```csharp  
// CS2016.cs  
// compile with: /codepage:x  
// CS2016 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
