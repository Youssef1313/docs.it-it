# <a name="compiler-error-cs0008"></a>Errore del compilatore CS0008
Errore imprevisto durante la lettura dei metadati dal file 'file' — 'description'  
  
 Una DLL aperta per il recupero dei metadati è danneggiata e non è stato possibile leggere i dati al suo interno. Per ulteriori informazioni, vedere [-Reference (C# opzioni del compilatore)](../language-reference/compiler-options/reference-compiler-option.md).
