# <a name="compiler-error-cs0161"></a>Errore del compilatore CS0161
'method': non tutti i percorsi del codice restituiscono un valore  
  
 Un metodo che restituisce un valore deve avere un'istruzione `return` in tutti i percorsi del codice. Per altre informazioni, vedere [Metodi](../programming-guide/classes-and-structs/methods.md).  
  
 L'esempio seguente genera l'errore CS0161:  
  
```csharp  
// CS0161.cs  
public class Test  
{  
   public static int Main() // CS0161  
   {  
      int i = 10;  
      if (i < 10)  
      {  
         return i;  
      }  
      else  
      {  
         // uncomment the following line to resolve  
         // return 1;  
      }  
   }  
}  
```
