# <a name="compiler-error-cs1510"></a>Errore del compilatore CS1510
Un argomento ref o out deve essere una variabile assegnabile  
  
 Solo una variabile pu√≤ essere passata come parametro [ref](../language-reference/keywords/ref.md) in una chiamata al metodo. Un valore `ref` corrisponde a passare un puntatore.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1510:  
  
```csharp  
// CS1510.cs  
public class C  
{  
   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
