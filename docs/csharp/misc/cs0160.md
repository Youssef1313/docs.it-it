# <a name="compiler-error-cs0160"></a><span data-ttu-id="ec00f-102">Errore del compilatore CS0160</span><span class="sxs-lookup"><span data-stu-id="ec00f-102">Compiler Error CS0160</span></span>
<span data-ttu-id="ec00f-103">Una clausola catch precedente rileva già tutte le eccezioni del tipo this o super ('type')</span><span class="sxs-lookup"><span data-stu-id="ec00f-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="ec00f-104">Una serie di istruzioni [catch](../language-reference/keywords/try-catch.md) deve essere in ordine di derivazione decrescente.</span><span class="sxs-lookup"><span data-stu-id="ec00f-104">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="ec00f-105">Ad esempio, gli oggetti più derivati devono essere visualizzati per prima.</span><span class="sxs-lookup"><span data-stu-id="ec00f-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="ec00f-106">Per ulteriori informazioni, vedere [eccezioni e gestione delle eccezioni](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="ec00f-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="ec00f-107">L'esempio seguente genera l'errore CS0160:</span><span class="sxs-lookup"><span data-stu-id="ec00f-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
