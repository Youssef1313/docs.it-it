# <a name="compiler-error-cs1593"></a>Errore del compilatore CS1593
Il delegato 'del' non accetta argomenti 'number'  
  
 Il numero di argomenti passati alla chiamata di un [delegato](../language-reference/keywords/delegate.md) non corrisponde al numero di parametri presenti nella dichiarazione del delegato.  
  
 L'esempio seguente genera l'errore CS1593:  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
