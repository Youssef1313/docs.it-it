# <a name="compiler-error-cs1900"></a>Errore del compilatore CS1900
Il livello di avviso deve essere compreso tra 0 e 4  
  
 L'opzione del compilatore [/warn](../language-reference/compiler-options/warn-compiler-option.md) può accettare solo uno dei cinque valori possibili (0, 1, 2, 3 o 4). Qualsiasi altro valore passato a **/warn** genererà l'errore CS1900.  
  
 L'esempio seguente genera l'errore CS1900:  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
