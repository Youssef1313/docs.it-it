# <a name="compiler-warning-level-1-cs2023"></a>Avviso del compilatore (livello 1) CS2023
Opzione /noconfig ignorata perché specificata in un file di risposta  
  
 L'opzione del compilatore [/noconfig](../language-reference/compiler-options/noconfig-compiler-option.md) è stata specificata in un file di risposta. Questa operazione non è consentita.  
  
 Questo avviso non può essere eliminato con l'opzione [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) .
