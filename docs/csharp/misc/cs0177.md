# <a name="compiler-error-cs0177"></a>Errore del compilatore CS0177
Il parametro out 'parameter' deve essere assegnato prima che il controllo lasci il metodo corrente  
  
 A un parametro contrassegnato con la parola chiave [out](../language-reference/keywords/out-parameter-modifier.md) non Ã¨ stato assegnato un valore nel corpo del metodo. Per ulteriori informazioni, vedere [passaggio di parametri](../programming-guide/classes-and-structs/passing-parameters.md)  
  
 L'esempio seguente genera l'errore CS0177:  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
