# <a name="compiler-error-cs1576"></a>Errore del compilatore CS1576
Il numero di riga specificato per la direttiva #line manca o non Ã¨ valido  
  
 Il compilatore ha rilevato un errore con il valore passato alla direttiva [#line](../language-reference/preprocessor-directives/preprocessor-line.md) .  
  
 L'esempio seguente genera l'errore CS1576:  
  
```csharp  
// CS1576.cs  
public class MyClass  
{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead  
      //#line  101 "abc.sc"  
      intt i;  // error will be reported on line 101  
   }  
}  
```
