# <a name="compiler-error-cs1520"></a><span data-ttu-id="5f76b-102">Errore del compilatore CS1520</span><span class="sxs-lookup"><span data-stu-id="5f76b-102">Compiler Error CS1520</span></span>
<span data-ttu-id="5f76b-103">Il metodo deve avere un tipo restituito</span><span class="sxs-lookup"><span data-stu-id="5f76b-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="5f76b-104">Un metodo dichiarato in una classe, in uno struct o in un'interfaccia deve includere un tipo restituito esplicito.</span><span class="sxs-lookup"><span data-stu-id="5f76b-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="5f76b-105">Nell'esempio seguente, il metodo Square ha un valore restituito [string](../language-reference/builtin-types/reference-types.md):</span><span class="sxs-lookup"><span data-stu-id="5f76b-105">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="5f76b-106">L'esempio seguente genera l'errore CS1520:</span><span class="sxs-lookup"><span data-stu-id="5f76b-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="5f76b-107">In alternativa, l'errore può verificarsi quando la combinazione di maiuscole e minuscole nel nome di un costruttore è diversa da quella della dichiarazione della classe o dello struct, come nell'esempio seguente:</span><span class="sxs-lookup"><span data-stu-id="5f76b-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="5f76b-108">Poiché il nome è non esattamente identico a quello della classe, il compilatore lo interpreta come un metodo normale, non come un costruttore, e genera l'errore:</span><span class="sxs-lookup"><span data-stu-id="5f76b-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
