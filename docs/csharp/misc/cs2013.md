# <a name="compiler-error-cs2013"></a>Errore del compilatore CS2013
'value' non è un numero di base dell'immagine valido  
  
 Un valore non valido (non numerico) è stato passato all'opzione del compilatore [/baseaddress](../language-reference/compiler-options/baseaddress-compiler-option.md) .  
  
 L'esempio seguente genera l'errore CS2013:  
  
```csharp  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
