# <a name="compiler-error-cs1621"></a>Errore del compilatore CS1621
Impossibile utilizzare l'istruzione yield all'interno di un metodo anonimo o di un'espressione lambda  
  
Non Ã¨ possibile usare l'istruzione [yield](../../csharp/language-reference/keywords/yield.md) in un [metodo anonimo](../language-reference/operators/delegate-operator.md) o un' [espressione lambda](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).
  
L'esempio seguente genera l'errore CS1621:
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
