# <a name="compiler-error-cs0629"></a><span data-ttu-id="7c2a0-102">Errore del compilatore CS0629</span><span class="sxs-lookup"><span data-stu-id="7c2a0-102">Compiler Error CS0629</span></span>
<span data-ttu-id="7c2a0-103">Il membro condizionale 'member' non può implementare il membro di interfaccia 'base class member' nel tipo 'Type Name'</span><span class="sxs-lookup"><span data-stu-id="7c2a0-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="7c2a0-104">L'attributo [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) non può essere usato nell'implementazione di un'interfaccia.</span><span class="sxs-lookup"><span data-stu-id="7c2a0-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="7c2a0-105">L'esempio seguente genera l'errore CS0629:</span><span class="sxs-lookup"><span data-stu-id="7c2a0-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
