# <a name="compiler-error-cs0643"></a>Errore del compilatore CS0643
L'argomento di attributo denominato 'arg' è duplicato  
  
 Un parametro, `arg`, in un attributo definito dall'utente è stato specificato due volte. Per altre informazioni, vedere [Attributi](../programming-guide/concepts/attributes/index.md).  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS0643:  
  
```csharp  
// CS0643.cs  
using System;  
using System.Runtime.InteropServices;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class MyAttribute : Attribute  
{  
    public MyAttribute()  
    {  
    }  
  
    public int x;  
}  
  
[MyAttribute(x = 5, x = 6)]   // CS0643, error setting x twice  
// try the following line instead  
// [MyAttribute(x = 5)]  
class MyClass  
{  
}  
  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```
