# <a name="compiler-error-cs0452"></a><span data-ttu-id="873a9-102">Errore del compilatore CS0452</span><span class="sxs-lookup"><span data-stu-id="873a9-102">Compiler Error CS0452</span></span>
<span data-ttu-id="873a9-103">Il tipo 'type name' deve essere un tipo riferimento per poter essere usato come parametro 'parameter name' nel metodo o nel tipo generico 'identifier of generic'</span><span class="sxs-lookup"><span data-stu-id="873a9-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="873a9-104">Questo errore si verifica quando si passa un tipo valore, ad esempio uno `struct` o `int` come parametro a un tipo generico o a un metodo che presenta un vincolo di tipo riferimento.</span><span class="sxs-lookup"><span data-stu-id="873a9-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="873a9-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="873a9-105">Example</span></span>  
 <span data-ttu-id="873a9-106">Il codice seguente genera l'errore CS0452.</span><span class="sxs-lookup"><span data-stu-id="873a9-106">The following code generates error CS0452.</span></span>

// CS0452.cs

public class Derived1 : BaseClass<int> { } // CS0452
