# <a name="compiler-error-cs0172"></a><span data-ttu-id="7fc04-102">Errore del compilatore CS0172</span><span class="sxs-lookup"><span data-stu-id="7fc04-102">Compiler Error CS0172</span></span>
<span data-ttu-id="7fc04-103">Non è possibile determinare il tipo di espressione condizionale perché 'type1' e 'type2' sono reciprocamente convertibili in modo implicito</span><span class="sxs-lookup"><span data-stu-id="7fc04-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
<span data-ttu-id="7fc04-104">In un'istruzione condizionale, è necessario essere in grado di convertire in modo implicito i tipi su `:` entrambi i lati del token.</span><span class="sxs-lookup"><span data-stu-id="7fc04-104">In a conditional statement, you must be able to implicitly convert the types on either side of the `:` token.</span></span> <span data-ttu-id="7fc04-105">Non è inoltre possibile eseguire conversioni implicite reciproche; è necessaria una sola conversione.</span><span class="sxs-lookup"><span data-stu-id="7fc04-105">Also, there cannot be mutual implicit conversions; you only need one conversion.</span></span>
  
<span data-ttu-id="7fc04-106">L'esempio seguente genera l'errore CS0172:</span><span class="sxs-lookup"><span data-stu-id="7fc04-106">The following sample generates CS0172:</span></span>
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
