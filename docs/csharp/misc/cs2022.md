# <a name="compiler-error-cs2022"></a>Errore del compilatore CS2022
Le opzioni '/out' e '/target' devono trovarsi prima dei nomi dei file di origine  
  
 Le opzioni del compilatore [/out (impostazione del nome del file di output)](../language-reference/compiler-options/out-compiler-option.md) e [/target (specifica del formato del file di output)](../language-reference/compiler-options/target-compiler-option.md) , quando sono specificate nella riga di comando, devono precedere i file del codice sorgente.
