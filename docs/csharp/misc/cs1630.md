# <a name="compiler-error-cs1630"></a>Errore del compilatore CS1630
Opzione 'option' non valida per /errorreport; specificare prompt, send, queue o none  
  
 L'opzione della riga di comando [-errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) deve essere seguita da **prompt**, **Send**, **queue**o **None**, specificando l'azione da intraprendere quando si verifica un errore interno del compilatore.
