# <a name="compiler-warning-level-4-cs0422"></a>Avviso del compilatore (livello 4) CS0422

L'opzione /incremental non è più supportata

 La compilazione incrementale ( **/incr** o **/Incremental**) non è supportata C# in 2,0.
 
> [!NOTE]
> Questo avviso del compilatore non viene più usato in Roslyn.
