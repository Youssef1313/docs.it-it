# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="a928e-102">Avviso del compilatore (livello 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="a928e-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="a928e-103">'member' è obsoleto</span><span class="sxs-lookup"><span data-stu-id="a928e-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="a928e-104">Progettazione classi ha contrassegnato un membro con l'attributo [Obsolete](../programming-guide/concepts/attributes/common-attributes.md#Obsolete) .</span><span class="sxs-lookup"><span data-stu-id="a928e-104">The class designer marked a member with the [Obsolete](../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute.</span></span> <span data-ttu-id="a928e-105">Ciò significa che il membro potrebbe non essere supportato in una versione futura della classe.</span><span class="sxs-lookup"><span data-stu-id="a928e-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="a928e-106">L'esempio seguente illustra come accedere a un membro obsoleto che genera l'errore CS0612:</span><span class="sxs-lookup"><span data-stu-id="a928e-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
