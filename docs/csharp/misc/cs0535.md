# <a name="compiler-error-cs0535"></a><span data-ttu-id="05153-102">Errore del compilatore CS0535</span><span class="sxs-lookup"><span data-stu-id="05153-102">Compiler Error CS0535</span></span>
<span data-ttu-id="05153-103">'class' non implementa il membro di interfaccia 'member'</span><span class="sxs-lookup"><span data-stu-id="05153-103">'class' does not implement interface member 'member'</span></span>  
  
 <span data-ttu-id="05153-104">Una [classe](../language-reference/keywords/class.md) è derivata da un' [interfaccia](../language-reference/keywords/interface.md), ma non ha implementato uno o più membri dell'interfaccia.</span><span class="sxs-lookup"><span data-stu-id="05153-104">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="05153-105">Una classe deve implementare tutti i membri delle interfacce da cui deriva oppure essere dichiarata `abstract`.</span><span class="sxs-lookup"><span data-stu-id="05153-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="05153-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="05153-106">Example</span></span>  
 <span data-ttu-id="05153-107">L'esempio seguente genera l'errore CS0535.</span><span class="sxs-lookup"><span data-stu-id="05153-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
