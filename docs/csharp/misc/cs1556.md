# <a name="compiler-error-cs1556"></a>Errore del compilatore CS1556
'construct' specificato per il metodo Main deve essere una classe o uno struct valido  
  
 All'opzione del compilatore [-Main](../language-reference/compiler-options/main-compiler-option.md) è stato passato un identificatore che non è un nome di classe.
