 <span data-ttu-id="544ee-104">Questo errore si verifica quando si tenta di modificare un membro di un campo che è di sola lettura in cui la modifica non è consentita.</span><span class="sxs-lookup"><span data-stu-id="544ee-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="544ee-105">Per risolvere l'errore, limitare le assegnazioni ai campi di sola lettura al costruttore o a un inizializzatore di variabile oppure rimuovere la parola chiave di sola lettura dalla dichiarazione del campo.</span><span class="sxs-lookup"><span data-stu-id="544ee-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>

## <a name="example"></a><span data-ttu-id="544ee-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="544ee-106">Example</span></span>

 <span data-ttu-id="544ee-107">L'esempio seguente genera l'errore CS1648:</span><span class="sxs-lookup"><span data-stu-id="544ee-107">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```
