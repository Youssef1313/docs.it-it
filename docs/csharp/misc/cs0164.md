# <a name="compiler-warning-level-2-cs0164"></a>Avviso del compilatore (livello 2) CS0164
Non è stato fatto riferimento a questa etichetta

 Un'etichetta è stata dichiarata, ma non è stata mai usata.

 L'esempio seguente genera l'errore CS0164:

```csharp
// CS0164.cs
// compile with: /W:2
public class Program
{
    public static void Main()
    {
        int i = 0;
    l1: // CS0164
        i++;
        // Uncomment the following lines to resolve this warning.
        // if (i < 10)
        //     goto l1;
    }
}
```
