 <span data-ttu-id="f4bec-104">Le espressioni lambda e i metodi anonimi non possono essere usati sul lato sinistro di [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) o [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator).</span><span class="sxs-lookup"><span data-stu-id="f4bec-104">Lambda expressions and anonymous methods may not be used on the left side of [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) or [as](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator).</span></span>
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f4bec-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="f4bec-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="f4bec-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="f4bec-109">Example</span></span>

<span data-ttu-id="f4bec-110">L'esempio seguente genera l'errore CS0837:</span><span class="sxs-lookup"><span data-stu-id="f4bec-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
