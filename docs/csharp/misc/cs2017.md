# <a name="compiler-error-cs2017"></a>Errore del compilatore CS2017
Non è possibile specificare /main se si compila un modulo o una libreria  
  
 Non è possibile specificare un punto di ingresso principale quando si compila [/target:library](../language-reference/compiler-options/target-library-compiler-option.md).  
  
 L'esempio seguente genera l'errore CS2017:  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
