## <a name="to-correct-this-error"></a><span data-ttu-id="60852-104">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="60852-104">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="60852-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="60852-107">Example</span></span>  
 <span data-ttu-id="60852-108">Il codice seguente genera l'errore CS0825 perché `var` viene usato in un campo di classe:</span><span class="sxs-lookup"><span data-stu-id="60852-108">The following code generates CS0825 because `var` is used on a class field:</span></span>  
  
```csharp  
// cs0825.cs  
class Test  
{  
    // Both of these declarations trigger CS0825
    private var genreName;   
    private var bookTitles = new List<string>();
  
    static int Main()  
    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```
