# <a name="compiler-error-cs1511"></a><span data-ttu-id="7c72b-102">Errore del compilatore CS1511</span><span class="sxs-lookup"><span data-stu-id="7c72b-102">Compiler Error CS1511</span></span>

 <span data-ttu-id="7c72b-104">La parola chiave [base](../language-reference/keywords/base.md) è stata usata in un metodo [static](../language-reference/keywords/static.md) .</span><span class="sxs-lookup"><span data-stu-id="7c72b-104">The [base](../language-reference/keywords/base.md) keyword was used in a [static](../language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="7c72b-105">`base` può essere chiamato solo in un costruttore di istanze, un metodo di istanza o una funzione di accesso di istanza.</span><span class="sxs-lookup"><span data-stu-id="7c72b-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7c72b-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="7c72b-106">Example</span></span>  
 <span data-ttu-id="7c72b-107">L'esempio seguente genera l'errore CS1511.</span><span class="sxs-lookup"><span data-stu-id="7c72b-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
