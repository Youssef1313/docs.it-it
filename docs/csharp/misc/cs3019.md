## <a name="example"></a><span data-ttu-id="cbce3-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="cbce3-107">Example</span></span>  
 <span data-ttu-id="cbce3-108">L'esempio seguente genera l'errore CS3019:</span><span class="sxs-lookup"><span data-stu-id="cbce3-108">The following sample generates CS3019:</span></span>

```csharp  
using System;  
  
[assembly: CLSCompliant(true)]

{  
    [CLSCompliant(false)]  // CS3019

    {  
    }

    {  
    }  
}  
```
