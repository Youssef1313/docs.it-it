## <a name="example"></a><span data-ttu-id="120e9-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="120e9-106">Example</span></span>  
 <span data-ttu-id="120e9-107">L'esempio seguente genera l'errore CS0154:</span><span class="sxs-lookup"><span data-stu-id="120e9-107">The following sample generates CS0154:</span></span>  
  
```csharp  
// CS0154.cs  
public class MyClass2  
{  
    public int i  
    {  
        set  
        {  
        }  
        // uncomment the get method to resolve this error  
        /*  
        get  
        {  
            return 0;  
        }  
        */  
    }  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClass2 myClass2 = new MyClass2();  
        int j = myClass2.i;   // CS0154, no get method  
    }  
}  
```
