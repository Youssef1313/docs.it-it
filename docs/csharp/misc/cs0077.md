# <a name="compiler-error-cs0077"></a>Errore del compilatore CS0077
L'operatore as deve essere usato con un tipo riferimento o con un tipo nullable ('int' è un tipo valore non nullable).  
  
 All'operatore [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operatore è stato passato un [tipo valore](../language-reference/keywords/value-types.md). Poiché `as` può restituire [null](../language-reference/keywords/null.md), è possibile passare solo [tipi riferimento](../language-reference/keywords/reference-types.md) o tipi di valore Nullable. Per ulteriori informazioni sui tipi di valore Nullable, vedere [tipi di valore Nullable](../programming-guide/nullable-types/index.md).
  
 L'esempio seguente genera l'errore CS0077:  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
