# <a name="compiler-error-cs2006"></a>Errore del compilatore CS2006
Errore nella sintassi della riga di comando: ' Text ' mancante per l'opzione ' Option '  
  
 La sintassi per *option* richiede testo aggiuntivo. Per informazioni, vedere [Opzioni del compilatore](../language-reference/compiler-options/index.md).
