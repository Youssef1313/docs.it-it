 <span data-ttu-id="93ed8-104">Un evento in un'interfaccia non può avere un inizializzatore.</span><span class="sxs-lookup"><span data-stu-id="93ed8-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="93ed8-105">Per altre informazioni, vedere [Interfacce](../programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="93ed8-105">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>

```csharp  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068

{  
   event MyDelegate d = new MyDelegate(M.f);

   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
