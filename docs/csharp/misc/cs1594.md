# <a name="compiler-error-cs1594"></a><span data-ttu-id="b2eb1-102">Errore del compilatore CS1594</span><span class="sxs-lookup"><span data-stu-id="b2eb1-102">Compiler Error CS1594</span></span>
<span data-ttu-id="b2eb1-103">Il delegato 'delegate' ha argomenti non validi</span><span class="sxs-lookup"><span data-stu-id="b2eb1-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="b2eb1-104">Il tipo di un argomento passato alla chiamata di un [delegato](../language-reference/keywords/delegate.md) non corrisponde al tipo del parametro nella dichiarazione del delegato.</span><span class="sxs-lookup"><span data-stu-id="b2eb1-104">The type of an argument passed to a [delegate](../language-reference/keywords/delegate.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="b2eb1-105">L'esempio seguente genera l'errore CS1594:</span><span class="sxs-lookup"><span data-stu-id="b2eb1-105">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
