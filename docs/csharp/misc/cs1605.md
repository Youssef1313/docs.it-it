# <a name="compiler-error-cs1605"></a>Errore del compilatore CS1605
Non è possibile passare 'var' come argomento ref o out perché è di sola lettura  
  
 Una variabile passata come parametro [ref](../language-reference/keywords/ref.md) o [out](../language-reference/keywords/out-parameter-modifier.md) deve essere modificata nel metodo chiamato. Non è quindi possibile passare un parametro di sola lettura come `ref` o `out`.
