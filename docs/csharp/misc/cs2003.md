# <a name="compiler-error-cs2003"></a>Errore del compilatore CS2003
Il file di risposta 'file' è incluso più volte  
  
 Un file di risposta è stato passato al compilatore più volte. I file di risposta possono essere passati al compilatore una sola volta per ogni file di output.  
  
 Per altre informazioni sui file di risposta, vedere [@ (specifica di un file di risposta)](../language-reference/compiler-options/response-file-compiler-option.md).
