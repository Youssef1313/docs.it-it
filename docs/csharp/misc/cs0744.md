# <a name="compiler-error-cs0744"></a><span data-ttu-id="36978-102">Errore del compilatore CS0744</span><span class="sxs-lookup"><span data-stu-id="36978-102">Compiler Error CS0744</span></span>
<span data-ttu-id="36978-103">È prevista la parola chiave contestuale 'equals'</span><span class="sxs-lookup"><span data-stu-id="36978-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="36978-104">Il percorso per una clausola `join` è `join`...`in`...`on`...`equals`, come mostrato nell'esempio:</span><span class="sxs-lookup"><span data-stu-id="36978-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
            join y in array2 on x equals y  
            select x;  
```

1. <span data-ttu-id="36978-106">Aggiungere la parola chiave `equals` alla clausola `join` .</span><span class="sxs-lookup"><span data-stu-id="36978-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="36978-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="36978-107">Example</span></span>  
 <span data-ttu-id="36978-108">Il codice seguente genera l'errore CS0744:</span><span class="sxs-lookup"><span data-stu-id="36978-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
