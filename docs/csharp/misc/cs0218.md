# <a name="compiler-error-cs0218"></a>Errore del compilatore CS0218
Il tipo ('type') deve contenere dichiarazioni di operatore true e operatore false  
  
Se un tipo definito dall'utente consente di eseguire l'overload dell'operatore [ &#124; ](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)o dell'operatore di [&](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) , Ã¨ necessario definire anche gli operatori [true e false](../language-reference/operators/true-false-operators.md) per fare in modo che il cortocircuito [& & operatore](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) o [ &#124; &#124; operatore](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) definito.
  
 L'esempio seguente genera l'errore CS0218:  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
