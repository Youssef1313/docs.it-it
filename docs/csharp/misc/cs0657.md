 Il compilatore ha rilevato un modificatore di attributo in una posizione non valida. Per altre informazioni, vedere [Destinazioni degli attributi](../programming-guide/concepts/attributes/index.md#attribute-targets) .  
  
 L'esempio seguente genera l'errore CS0657:  
  
```csharp  
// CS0657.cs  
// compile with: /target:library  
public class TestAttribute : System.Attribute {}  
[return: Test]   // CS0657 return not valid on a class  
class Class1 {}  
```
