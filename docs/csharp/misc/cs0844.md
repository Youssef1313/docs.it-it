## <a name="example"></a><span data-ttu-id="2af5e-111">Esempio</span><span class="sxs-lookup"><span data-stu-id="2af5e-111">Example</span></span>  
 <span data-ttu-id="2af5e-112">Il codice seguente genera l'errore CS0844:</span><span class="sxs-lookup"><span data-stu-id="2af5e-112">The following code generates CS0844:</span></span>  
  
```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="2af5e-113">Correggere l'errore tramite `this.num` per fare riferimento al campo della classe</span><span class="sxs-lookup"><span data-stu-id="2af5e-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
