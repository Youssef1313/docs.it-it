<span data-ttu-id="e32d7-103">Impossibile utilizzare l'istruzione fixed per accettare l'indirizzo di un'espressione già di tipo fixed</span><span class="sxs-lookup"><span data-stu-id="e32d7-103">You cannot use the fixed statement to take the address of an already fixed expression</span></span>  
  
 <span data-ttu-id="e32d7-104">Una variabile locale in un metodo [unsafe](../language-reference/keywords/unsafe.md) o in un parametro è già di tipo fixed (nello stack), pertanto non è possibile accettare l'indirizzo di una di queste due variabili in un'espressione di tipo [fixed](../language-reference/keywords/fixed-statement.md) .</span><span class="sxs-lookup"><span data-stu-id="e32d7-104">A local variable in an [unsafe](../language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="e32d7-105">Per altre informazioni, vedere [Codice unsafe e puntatori](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="e32d7-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="e32d7-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="e32d7-106">Example</span></span>  
 <span data-ttu-id="e32d7-107">L'esempio seguente genera l'errore CS0213.</span><span class="sxs-lookup"><span data-stu-id="e32d7-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
