# <a name="compiler-error-cs0040"></a>Errore del compilatore CS0040
Errore imprevisto durante la creazione del file di informazioni di debug: 'reason'  
  
 Questo errore può verificarsi quando si usa l'opzione del compilatore [-debug](../language-reference/compiler-options/debug-compiler-option.md) e indica che il compilatore non è riuscito a scrivere nel file PDB. Per risolvere l'errore è possibile reinstallare Visual Studio, assicurandosi che il compilatore abbia accesso in scrittura a un file o una directory, oppure non usare l'opzione /debug nella compilazione.
