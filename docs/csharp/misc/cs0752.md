# <a name="compiler-error-cs0752"></a><span data-ttu-id="c7613-102">Errore del compilatore CS0752</span><span class="sxs-lookup"><span data-stu-id="c7613-102">Compiler Error CS0752</span></span>
<span data-ttu-id="c7613-103">Un metodo parziale non può avere parametri out</span><span class="sxs-lookup"><span data-stu-id="c7613-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="c7613-104">Un metodo parziale non può avere un parametro [out](../language-reference/keywords/out-parameter-modifier.md) .</span><span class="sxs-lookup"><span data-stu-id="c7613-104">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="c7613-105">I parametri out non sono consentiti perché se il metodo parziale viene rimosso dal compilatore, non è garantito che il parametro out venga assegnato.</span><span class="sxs-lookup"><span data-stu-id="c7613-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c7613-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="c7613-106">To correct this error</span></span>

    }
