# <a name="compiler-error-cs0690"></a>Errore del compilatore CS0690
Il file di input 'file' contiene metadati non validi.  
  
 È possibile aprire il file di metadati ma, a causa di un problema di localizzazione, i metadati sono stati danneggiati. Si tratta di una situazione analoga all'errore all'errore [CS0009](./cs0009.md), ad eccezione del fatto che è possibile aprire il file di metadati.  
  
 Un file PE (eseguibile del processo) è un file eseguibile.
