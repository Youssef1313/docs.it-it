 <span data-ttu-id="dd780-104">Il tentativo di eseguire l'override di un metodo della funzione di accesso a una proprietà non è riuscito perché non è possibile eseguire l'override della funzione di accesso.</span><span class="sxs-lookup"><span data-stu-id="dd780-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="dd780-105">Questo errore si verifica nei seguenti casi:</span><span class="sxs-lookup"><span data-stu-id="dd780-105">This error can occur if:</span></span>

## <a name="example"></a><span data-ttu-id="dd780-110">Esempio</span><span class="sxs-lookup"><span data-stu-id="dd780-110">Example</span></span>  
 <span data-ttu-id="dd780-111">L'esempio seguente genera l'errore CS0546 perché la classe base non dichiara una funzione di accesso set per la proprietà.</span><span class="sxs-lookup"><span data-stu-id="dd780-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
