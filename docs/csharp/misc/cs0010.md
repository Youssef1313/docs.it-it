# <a name="compiler-error-cs0010"></a>Errore del compilatore CS0010
Errore irreversibile imprevisto -- 'error'.  
  
 Questo errore viene generato quando si verifica un evento completamente imprevisto che arresta la compilazione.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Ricompilare un altro progetto. Se viene visualizzato lo stesso errore, provare a reinstallare Visual Studio. Se viene visualizzato questo errore, inviare una segnalazione errori a Microsoft.
