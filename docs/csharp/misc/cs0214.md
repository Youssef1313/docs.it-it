 <span data-ttu-id="ca4e8-104">I puntatori possono essere usati solo la parola chiave [unsafe](../language-reference/keywords/unsafe.md) .</span><span class="sxs-lookup"><span data-stu-id="ca4e8-104">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="ca4e8-105">Per altre informazioni, vedere [Codice unsafe e puntatori](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="ca4e8-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="ca4e8-106">L'esempio seguente genera l'errore CS0214:</span><span class="sxs-lookup"><span data-stu-id="ca4e8-106">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
