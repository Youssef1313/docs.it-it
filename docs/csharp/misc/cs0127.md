# <a name="compiler-error-cs0127"></a><span data-ttu-id="329e7-102">Errore del compilatore CS0127</span><span class="sxs-lookup"><span data-stu-id="329e7-102">Compiler Error CS0127</span></span>
<span data-ttu-id="329e7-103">Poiché 'function' restituisce void, una parola chiave restituita non deve essere seguita da un'espressione dell'oggetto</span><span class="sxs-lookup"><span data-stu-id="329e7-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="329e7-104">Un metodo con un tipo restituito [void](../language-reference/keywords/void.md) non può restituire un valore.</span><span class="sxs-lookup"><span data-stu-id="329e7-104">A method with a [void](../language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="329e7-105">Per altre informazioni, vedere [Metodi](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="329e7-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="329e7-106">L'esempio seguente genera l'errore CS0127:</span><span class="sxs-lookup"><span data-stu-id="329e7-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
