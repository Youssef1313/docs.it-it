# <a name="compiler-error-cs1515"></a>Errore del compilatore CS1515
È previsto 'in'  
  
 In un'istruzione [foreach, in](../language-reference/keywords/foreach-in.md) , la parte "in" è mancante.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1515:  
  
```csharp 
using System;  
  
class Driver  
{  
   static void Main()  
   {  
      int[] arr = new int[] {1, 2, 3};  
  
      // try the following line instead  
      // foreach (int x in arr)  
      foreach (int x arr)      // CS1515, "in" is missing  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
