# <a name="compiler-error-cs0573"></a>Errore del compilatore CS0573
'field declaration': impossibile inizializzare il campo di un'istanza negli struct  
  
 Non Ã¨ possibile inizializzare un campo di istanza di uno [struct](../language-reference/keywords/struct.md). I campi di tipi valore verranno inizializzati sui relativi valori predefiniti e i campi di tipo riferimento verranno inizializzati su `null`.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS0573:  
  
```csharp  
// CS0573.cs  
namespace x  
{  
    public class clx  
    {  
        public static void Main()  
        {  
        }  
    }  
  
    public struct cly  
    {  
        clx a = new clx();   // CS0573  
        // clx a;            // OK  
        int i = 7;           // CS0573  
        // int i;            // OK  
    }  
}  
```
