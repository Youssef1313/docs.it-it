# <a name="compiler-error-cs0215"></a>Errore del compilatore CS0215
Il tipo restituito dell'operatore True o False deve essere booleano  
  
Gli operatori [true e false](../language-reference/operators/true-false-operators.md) definiti dall'utente devono avere un tipo restituito [bool](../language-reference/builtin-types/bool.md).
  
L'esempio seguente genera l'errore CS0215:  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
