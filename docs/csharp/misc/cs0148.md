# <a name="compiler-error-cs0148"></a>Errore del compilatore CS0148
Il delegato 'delegato' non presenta un costruttore valido  
  
 È stato importato e usato un programma (che usa Common Language Runtime di .NET Framework) creato con un altro compilatore. Il compilatore ha consentito un costruttore [delegate](../language-reference/keywords/delegate.md) il cui formato non è corretto. Per altre informazioni, vedere [Delegati](../programming-guide/delegates/index.md).
