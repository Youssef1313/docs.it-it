# <a name="compiler-error-cs1527"></a><span data-ttu-id="465ae-102">Errore del compilatore CS1527</span><span class="sxs-lookup"><span data-stu-id="465ae-102">Compiler Error CS1527</span></span>

<span data-ttu-id="465ae-103">Gli elementi definiti in uno spazio dei nomi non possono essere dichiarati in modo esplicito come privato, protetto, protetto interno o privato protetto.</span><span class="sxs-lookup"><span data-stu-id="465ae-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="465ae-104">Le dichiarazioni di tipo in uno spazio dei nomi possono avere accesso [public](../../csharp/language-reference/keywords/public.md) o [internal](../../csharp/language-reference/keywords/internal.md) .</span><span class="sxs-lookup"><span data-stu-id="465ae-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="465ae-105">Se non è specificata alcuna accessibilità, l'impostazione predefinita è **internal** .</span><span class="sxs-lookup"><span data-stu-id="465ae-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="465ae-106">L'esempio seguente genera l'errore CS1527:</span><span class="sxs-lookup"><span data-stu-id="465ae-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="465ae-107">L'esempio seguente genera l'errore CS1527 perché quando nessuno spazio dei nomi è dichiarato in modo esplicito nel codice programma, tutte le dichiarazioni di tipo si trovano in modo implicito nello spazio dei nomi globale.</span><span class="sxs-lookup"><span data-stu-id="465ae-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
