# <a name="compiler-error-cs0244"></a>Errore del compilatore CS0244
is' o 'as' non valido per tipi puntatore  
  
 Gli operatori [is](../../csharp/language-reference/operators/type-testing-and-cast.md#is-operator) e [As](../../csharp/language-reference/operators/type-testing-and-cast.md#as-operator) non sono validi per l'utilizzo nei tipi di puntatore. Per altre informazioni, vedere [Codice unsafe e puntatori](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 L'esempio seguente genera l'errore CS0244:  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
