# <a name="compiler-error-cs0763"></a><span data-ttu-id="2759e-102">Errore del compilatore CS0763</span><span class="sxs-lookup"><span data-stu-id="2759e-102">Compiler Error CS0763</span></span>
<span data-ttu-id="2759e-103">Entrambe le dichiarazioni di metodo parziale devono essere statiche, altrimenti nessuna delle due potrà esserlo.</span><span class="sxs-lookup"><span data-stu-id="2759e-103">Both partial method declarations must be static or neither may be static.</span></span>  
  
 <span data-ttu-id="2759e-104">Una dichiarazione di metodo parziale non può avere una parte statica e una parte non statica.</span><span class="sxs-lookup"><span data-stu-id="2759e-104">A partial method declaration cannot have one part static and the other part not static.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2759e-105">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="2759e-105">To correct this error</span></span>  
  
1. <span data-ttu-id="2759e-106">Rendere entrambe le parti statiche o non statiche.</span><span class="sxs-lookup"><span data-stu-id="2759e-106">Make both parts either static or non-static.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2759e-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="2759e-107">Example</span></span>  
 <span data-ttu-id="2759e-108">Il codice seguente genera l'errore CS0763:</span><span class="sxs-lookup"><span data-stu-id="2759e-108">The following code generates CS0763:</span></span>  
  
```csharp  
// cs0763.cs  
using System;  
  
    public partial class C  
    {  
        static partial void Part();  
        partial void Part() // CS0763  
        {  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
  
    }  
```
