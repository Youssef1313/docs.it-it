## <a name="example"></a><span data-ttu-id="c004b-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="c004b-108">Example</span></span>  
 <span data-ttu-id="c004b-109">Il codice seguente genera l'errore CS0314:</span><span class="sxs-lookup"><span data-stu-id="c004b-109">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
