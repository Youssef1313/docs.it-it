# <a name="compiler-error-cs2011"></a>Errore del compilatore CS2011
Non è possibile aprire il file di risposta 'file'  
  
 È stato specificato un file di risposta in una compilazione, ma il compilatore non è riuscito a trovare e aprire il file.  
  
 Per altre informazioni sui file di risposta, vedere [@ (specifica di un file di risposta)](../language-reference/compiler-options/response-file-compiler-option.md).
