# <a name="compiler-error-cs1557"></a>Errore del compilatore CS1557
Impossibile utilizzare 'class' per il metodo Main perché si trova in un file di output diverso  
  
 È stata specificata l'opzione del compilatore [/main](../language-reference/compiler-options/main-compiler-option.md) per un file di output in una compilazione di file multi-output. La classe, però, non è stata trovata nel codice sorgente per la compilazione /main, ma è stata trovata in un file di codice sorgente per uno degli altri file di output nella compilazione.
