# <a name="compiler-error-cs1559"></a>Errore del compilatore CS1559
Non è possibile usare 'object' per il metodo Main perché è importato  
  
 È stata specificata una classe non valida per l'opzione del compilatore [-Main](../language-reference/compiler-options/main-compiler-option.md) . la classe non può essere usata come posizione per il metodo [Main](../programming-guide/main-and-command-args/index.md) .
