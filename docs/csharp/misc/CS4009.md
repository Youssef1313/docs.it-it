# <a name="compiler-error-cs4009"></a>Errore del compilatore CS4009

' Type. Method ': Impossibile contrassegnare un punto di ingresso con `async` il modificatore.

Non è possibile usare `async` la parola chiave nel punto di ingresso dell'applicazione `Main` (in genere il metodo).  

> [!IMPORTANT]
> A partire C# da 7,1, `Main` il metodo può avere `async` un modificatore. Per ulteriori informazioni, vedere la sezione [principale asincrona](../whats-new/csharp-7-1.md#async-main) dell'articolo [Novità C# in 7,1](../whats-new/csharp-7-1.md) . Per informazioni su come selezionare la C# versione della lingua, vedere l'articolo selezionare la [versione della C# lingua](../language-reference/configure-language-version.md) .

L'esempio seguente produce CS4009:

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```

## <a name="to-correct-this-error"></a>Per correggere l'errore

[Aggiornare la C# versione della lingua](../language-reference/configure-language-version.md) utilizzata dal progetto a 7,1 o successiva.

Se si usa C# 7,0 o un valore inferiore, `async` rimuovere la parola chiave dalla firma del punto di ingresso dell'applicazione.  Rimuovere anche eventuali `await` parole chiave usate per attendere i metodi asincroni nel punto di ingresso dell'applicazione. 

Tuttavia, è comunque necessario attendere il completamento del metodo asincrono prima che il punto di ingresso riprenda l'esecuzione. In caso contrario, la compilazione genera l'avviso del compilatore CS4014 e l'applicazione verrà terminata prima del completamento dell'operazione asincrona.
