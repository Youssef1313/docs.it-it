# <a name="compiler-error-cs1038"></a>Errore del compilatore CS1038
\#Ã¨ prevista la direttiva endregion  
  
 Una direttiva [#region](../language-reference/preprocessor-directives/preprocessor-region.md) non ha una direttiva [#endregion](../language-reference/preprocessor-directives/preprocessor-endregion.md) corrispondente.  
  
 L'esempio seguente genera l'errore CS1038:  
  
```csharp  
// CS1038.cs  
#region testing  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
// CS1038  
// uncomment the next line to resolve  
// #endregion  
```
