# <a name="compiler-error-cs0041"></a>Errore del compilatore CS0041
Il nome completo per 'type' è troppo lungo per le informazioni di debug. Compilare senza l'opzione '/debug'.  
  
 Questo errore può verificarsi quando si usa l'opzione [/debug](../language-reference/compiler-options/debug-compiler-option.md) del compilatore. Se si verifica questo errore, provare a eliminare i file PDB nella directory bin e ricompilare. Se si sta ancora riscontrando questo errore, potrebbe essere necessario ripristinare o reinstallare Visual Studio.
