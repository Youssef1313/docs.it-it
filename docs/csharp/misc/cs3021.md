 <span data-ttu-id="fd538-104">Questo avviso viene generato `[CLSCompliant(false)]` se viene visualizzato in una classe in un assembly che non dispone di un attributo CLSCompliant a livello di assembly impostato su true (ovvero la `[assembly: CLSCompliant(true)]`riga).</span><span class="sxs-lookup"><span data-stu-id="fd538-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="fd538-105">Poiché l'assembly non dichiara se stesso come conforme a CLS, non è necessario che alcun elemento all'interno dell'assembly si dichiari non conforme, poiché previsto che non sia conforme a CLS.</span><span class="sxs-lookup"><span data-stu-id="fd538-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="fd538-106">Per altre informazioni sulla conformità a CLS, vedere [indipendenza del linguaggio e componenti indipendenti dal linguaggio](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="fd538-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>

## <a name="example"></a><span data-ttu-id="fd538-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="fd538-108">Example</span></span>  
 <span data-ttu-id="fd538-109">L'esempio seguente genera l'errore CS3021:</span><span class="sxs-lookup"><span data-stu-id="fd538-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
