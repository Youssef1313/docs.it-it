# <a name="compiler-warning-level-2-cs3021"></a>Avviso del compilatore (livello 2) CS3021
'type' non necessita di un attributo CLSCompliant perché l'assembly non ha un attributo CLSCompliant  
  
 Questo avviso viene generato `[CLSCompliant(false)]` se viene visualizzato in una classe in un assembly che non dispone di un attributo CLSCompliant a livello di assembly impostato su true (ovvero la `[assembly: CLSCompliant(true)]`riga). Poiché l'assembly non dichiara se stesso come conforme a CLS, non è necessario che alcun elemento all'interno dell'assembly si dichiari non conforme, poiché previsto che non sia conforme a CLS. Per altre informazioni sulla conformità a CLS, vedere [indipendenza del linguaggio e componenti indipendenti dal linguaggio](../../standard/language-independence.md).
  
 Per eliminare l'avviso, rimuovere l'attributo o aggiungere l'attributo a livello di assembly.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS3021:  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
