# <a name="compiler-error-cs0765"></a><span data-ttu-id="7d5b2-102">Errore del compilatore CS0765</span><span class="sxs-lookup"><span data-stu-id="7d5b2-102">Compiler Error CS0765</span></span>
<span data-ttu-id="7d5b2-103">Non è possibile usare negli alberi delle espressioni metodi parziali contenenti solo una dichiarazione di definizione o metodi condizionali rimossi</span><span class="sxs-lookup"><span data-stu-id="7d5b2-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="7d5b2-104">Anche se una chiamata a un metodo parziale rimosso è un'espressione, non è un'espressione accettabile in un albero delle espressioni.</span><span class="sxs-lookup"><span data-stu-id="7d5b2-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>

## <a name="example"></a><span data-ttu-id="7d5b2-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="7d5b2-107">Example</span></span>  
 <span data-ttu-id="7d5b2-108">Il codice seguente genera l'errore CS0765 in due posizioni:</span><span class="sxs-lookup"><span data-stu-id="7d5b2-108">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
