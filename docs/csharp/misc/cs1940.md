# <a name="compiler-error-cs1940"></a><span data-ttu-id="c93c3-102">Errore del compilatore CS1940</span><span class="sxs-lookup"><span data-stu-id="c93c3-102">Compiler Error CS1940</span></span>
<span data-ttu-id="c93c3-103">Sono state trovate più implementazioni del modello di query per il tipo di origine 'type'.</span><span class="sxs-lookup"><span data-stu-id="c93c3-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="c93c3-104">Chiamata ambigua a 'method'.</span><span class="sxs-lookup"><span data-stu-id="c93c3-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="c93c3-105">Questo errore viene generato quando vengono definite più implementazioni di un metodo di query e il compilatore non è in grado di distinguere quale sia quella più adatta per la query.</span><span class="sxs-lookup"><span data-stu-id="c93c3-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="c93c3-106">Nell'esempio seguente le due versioni di `Select` hanno la stessa firma perché entrambe accettano `int` come parametro di input e hanno `int` come valore restituito.</span><span class="sxs-lookup"><span data-stu-id="c93c3-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

## <a name="example"></a><span data-ttu-id="c93c3-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="c93c3-109">Example</span></span>  
 <span data-ttu-id="c93c3-110">Il codice seguente genera l'errore CS1940:</span><span class="sxs-lookup"><span data-stu-id="c93c3-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
