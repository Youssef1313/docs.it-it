# <a name="compiler-error-cs0538"></a><span data-ttu-id="58759-102">Errore del compilatore CS0538</span><span class="sxs-lookup"><span data-stu-id="58759-102">Compiler Error CS0538</span></span>
<span data-ttu-id="58759-103">'name' nella dichiarazione esplicita dell'interfaccia non è un'interfaccia</span><span class="sxs-lookup"><span data-stu-id="58759-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="58759-104">Si è provato a dichiarare esplicitamente un' [interfaccia](../language-reference/keywords/interface.md)che non è stata specificata.</span><span class="sxs-lookup"><span data-stu-id="58759-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="58759-105">L'esempio seguente genera l'errore CS0538:</span><span class="sxs-lookup"><span data-stu-id="58759-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
