# <a name="compiler-error-cs0155"></a>Errore del compilatore CS0155
Il tipo rilevato o generato deve derivare da System.Exception  
  
 Si Ã¨ provato a passare un tipo di dati che non deriva da **System.Exception** in un blocco [catch](../language-reference/keywords/try-catch.md) . Solo i tipi di dati che derivano da **System.Exception** possono essere passati in un blocco **catch** . Per ulteriori informazioni, vedere [eccezioni e gestione delle eccezioni](../programming-guide/exceptions/index.md).  
  
 L'esempio seguente genera l'errore CS0155:  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
