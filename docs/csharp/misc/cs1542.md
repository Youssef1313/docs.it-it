# <a name="compiler-error-cs1542"></a>Errore del compilatore CS1542
Impossibile aggiungere 'dll' a questo assembly perché si tratta già di assembly. Utilizzare l'opzione '/R'  
  
 Il file a cui si fa riferimento con l'opzione del compilatore [/addmodule](../language-reference/compiler-options/addmodule-compiler-option.md) non è stato creato con [/target:module](../language-reference/compiler-options/target-module-compiler-option.md). Usare [/reference](../language-reference/compiler-options/reference-compiler-option.md) per fare riferimento al file in questa compilazione.
