# <a name="compiler-error-cs0617"></a><span data-ttu-id="325d2-102">Errore del compilatore CS0617</span><span class="sxs-lookup"><span data-stu-id="325d2-102">Compiler Error CS0617</span></span>
<span data-ttu-id="325d2-103">'reference' non è un argomento di attributo denominato valido perché non è un tipo di parametro di attributo valido</span><span class="sxs-lookup"><span data-stu-id="325d2-103">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="325d2-104">Si è provato ad accedere a un membro [privato](../language-reference/keywords/private.md) di un attributo.</span><span class="sxs-lookup"><span data-stu-id="325d2-104">An attempt was made to access a [private](../language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="325d2-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="325d2-105">Example</span></span>  
 <span data-ttu-id="325d2-106">L'esempio seguente genera l'errore CS0617.</span><span class="sxs-lookup"><span data-stu-id="325d2-106">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |   
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
