# <a name="compiler-error-cs0030"></a>Errore del compilatore CS0030
Non è possibile convertire il tipo 'type' in 'type'  
  
Non viene eseguita alcuna conversione predefinita tra i tipi. È possibile definire una conversione personalizzata tra tali tipi. Per altre informazioni, vedere [gli operatori di conversione definita dall'utente](../../csharp/language-reference/operators/user-defined-conversion-operators.md).  
  
 L'esempio seguente genera l'errore CS0030:  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
