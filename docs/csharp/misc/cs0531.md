# <a name="compiler-error-cs0531"></a><span data-ttu-id="f054a-102">Errore del compilatore CS0531</span><span class="sxs-lookup"><span data-stu-id="f054a-102">Compiler Error CS0531</span></span>
<span data-ttu-id="f054a-103">'member': i membri di interfaccia non possono avere una definizione</span><span class="sxs-lookup"><span data-stu-id="f054a-103">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="f054a-104">I metodi dichiarati in un' [interfaccia](../language-reference/keywords/interface.md) devono essere implementati in una classe che eredita da questa e non nell'interfaccia stessa.</span><span class="sxs-lookup"><span data-stu-id="f054a-104">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="f054a-105">L'esempio seguente genera l'errore CS0531:</span><span class="sxs-lookup"><span data-stu-id="f054a-105">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
