# <a name="compiler-error-cs1906"></a>Errore del compilatore CS1906
L'opzione 'option' non è valida. La visibilità della risorsa deve essere 'public' o 'private'  
  
 Questo errore indica un'opzione della riga di comando [/resource (incorporamento del file di risorse nell'output)](../language-reference/compiler-options/resource-compiler-option.md) o [/linkresource (collegamento a risorsa .NET Framework)](../language-reference/compiler-options/linkresource-compiler-option.md) non valida. Controllare la sintassi dell'opzione della riga di comando **/resource** o **/linkresource** e verificare che il modificatore di accessibilità usato sia **public** o `private`.
