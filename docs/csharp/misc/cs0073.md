 <span data-ttu-id="d2f51-104">Una parola chiave **add** o **remove** in una definizione [event](../language-reference/keywords/event.md) deve avere un corpo.</span><span class="sxs-lookup"><span data-stu-id="d2f51-104">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="d2f51-105">Per ulteriori informazioni, vedi [Eventi](../programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="d2f51-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="d2f51-106">L'esempio seguente genera l'errore CS0073:</span><span class="sxs-lookup"><span data-stu-id="d2f51-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
