# <a name="compiler-error-cs0037"></a>Errore del compilatore CS0037
Non è possibile convertire Null in 'type' perché è un tipo valore non nullable  
  
 Il compilatore non può assegnare null a un tipo valore. null può essere assegnato solo a un [tipo riferimento](../language-reference/keywords/reference-types.md) o a un tipo Nullable. [struct](../language-reference/keywords/struct.md) è un tipo valore. Per altre informazioni, vedere [Tipi nullable](../programming-guide/nullable-types/index.md).  
  
 L'esempio seguente genera l'errore CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
