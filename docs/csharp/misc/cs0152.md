# <a name="compiler-error-cs0152"></a>Errore del compilatore CS0152
L'etichetta 'label' è già presente in questa istruzione switch  
  
 In un'istruzione [switch](../language-reference/keywords/switch.md) è stata ripetuta un'etichetta. Per altre informazioni, vedere [switch](../language-reference/keywords/switch.md).  
  
 L'esempio seguente genera l'errore CS0152:  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
