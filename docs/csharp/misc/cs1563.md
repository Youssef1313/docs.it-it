# <a name="compiler-error-cs1563"></a>Errore del compilatore CS1563
Per l'output 'file di output' non esistono file di origine  
  
 È stata specificata l'opzione [/out](../language-reference/compiler-options/out-compiler-option.md) del compilatore, ma non è presente alcun file di codice sorgente. Non usare l'opzione **/out**oppure specificare i file di codice sorgente dopo l'opzione **/out**.
