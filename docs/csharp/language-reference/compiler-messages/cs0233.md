# <a name="compiler-error-cs0233"></a>Errore del compilatore CS0233
'identificatore' non ha una dimensione predefinita, quindi sizeof può essere usato solo in un contesto di tipo unsafe
  
 Senza il contesto [unsafe](../keywords/unsafe.md), l'operatore [sizeof](../../../csharp/language-reference/operators/sizeof.md) può essere usato solo per i tipi la cui dimensione è una costante in fase di compilazione. Se si riceve questo errore, usare un contesto di tipo unsafe.
  
L'esempio seguente genera l'errore CS0233:
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
