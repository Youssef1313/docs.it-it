# <a name="compiler-error-cs0233"></a><span data-ttu-id="076ed-102">Errore del compilatore CS0233</span><span class="sxs-lookup"><span data-stu-id="076ed-102">Compiler Error CS0233</span></span>
<span data-ttu-id="076ed-103">'identificatore' non ha una dimensione predefinita, quindi sizeof può essere usato solo in un contesto di tipo unsafe</span><span class="sxs-lookup"><span data-stu-id="076ed-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="076ed-104">Senza il contesto [unsafe](../keywords/unsafe.md), l'operatore [sizeof](../operators/sizeof.md) può essere usato solo per i tipi la cui dimensione è una costante in fase di compilazione.</span><span class="sxs-lookup"><span data-stu-id="076ed-104">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="076ed-105">Se si riceve questo errore, usare un contesto di tipo unsafe.</span><span class="sxs-lookup"><span data-stu-id="076ed-105">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="076ed-106">L'esempio seguente genera l'errore CS0233:</span><span class="sxs-lookup"><span data-stu-id="076ed-106">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
