# <a name="compiler-error-cs1644"></a>Errore del compilatore CS1644
La funzionalità 'feature' non fa parte della specifica del linguaggio C# standard ISO e potrebbe non essere accettata da altri compilatori  
  
 Questo errore si verifica se è stata specificata l'opzione [/langversion](../compiler-options/langversion-compiler-option.md) ISO-1 e il codice da compilare usa funzionalità che non fanno parte dello standard ISO 1.0. Per correggere l'errore, non usare funzionalità del compilatore C# 2.0, ad esempio generics o metodi anonimi con l'opzione di compatibilità ISO-1.  
  
 L'esempio seguente genera l'errore CS1644:  
  
```csharp  
// CS1644.cs  
// compile with: /langversion:ISO-1 /target:library  
class C<T> {}   // CS1644  
```
