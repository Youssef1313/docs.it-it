# <a name="compiler-error-cs0579"></a><span data-ttu-id="d68ae-102">Errore del compilatore CS0579</span><span class="sxs-lookup"><span data-stu-id="d68ae-102">Compiler Error CS0579</span></span>
<span data-ttu-id="d68ae-103">Attributo "attribute" duplicato</span><span class="sxs-lookup"><span data-stu-id="d68ae-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="d68ae-104">Non è possibile specificare lo stesso attributo più di una volta, a meno che l'attributo specifichi **AllowMultiple = true** in [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).</span><span class="sxs-lookup"><span data-stu-id="d68ae-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="d68ae-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="d68ae-105">Example</span></span>  
 <span data-ttu-id="d68ae-106">L'esempio seguente genera l'errore CS0579.</span><span class="sxs-lookup"><span data-stu-id="d68ae-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
