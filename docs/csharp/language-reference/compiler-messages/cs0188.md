# <a name="compiler-error-cs0188"></a><span data-ttu-id="9713f-102">Errore del compilatore CS0188</span><span class="sxs-lookup"><span data-stu-id="9713f-102">Compiler Error CS0188</span></span>

<span data-ttu-id="9713f-103">Non è possibile usare l'oggetto 'this' prima dell'assegnazione di tutti i campi dell'oggetto</span><span class="sxs-lookup"><span data-stu-id="9713f-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="9713f-104">Tutti i campi in uno `struct` devono essere assegnati da un costruttore prima che il costruttore possa chiamare un metodo in `struct`.</span><span class="sxs-lookup"><span data-stu-id="9713f-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="9713f-105">Se questo errore viene visualizzato durante il tentativo di inizializzare una proprietà in un costruttore di struct, la soluzione consiste nel modificare il parametro del costruttore per specificare il campo sottostante anziché la proprietà stessa.</span><span class="sxs-lookup"><span data-stu-id="9713f-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="9713f-106">Le proprietà implementate automaticamente devono essere evitate negli struct perché non hanno un campo sottostante e pertanto non può essere inizializzate in alcun modo dal costruttore.</span><span class="sxs-lookup"><span data-stu-id="9713f-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>  
  
 <span data-ttu-id="9713f-107">Per altre informazioni, vedere [Uso di struct](../../programming-guide/classes-and-structs/using-structs.md).</span><span class="sxs-lookup"><span data-stu-id="9713f-107">For more information, see [Using Structs](../../programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="9713f-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="9713f-108">Example</span></span>

 <span data-ttu-id="9713f-109">L'esempio seguente genera l'errore CS0188:</span><span class="sxs-lookup"><span data-stu-id="9713f-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
