# <a name="compiler-error-cs0006"></a>Errore del compilatore CS0006

Il file dei metadati 'nome_dll' non è stato trovato
  
 Il programma è stato compilato ed è stato passato in modo esplicito il nome di un file contenente metadati. Tuttavia, il file DLL non è stato trovato. Per ulteriori informazioni, vedere [-Reference (C# opzioni del compilatore)](../compiler-options/reference-compiler-option.md).
