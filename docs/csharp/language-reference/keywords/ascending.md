# <a name="ascending-c-reference"></a>ascending (Riferimenti per C#)
La parola chiave contestuale `ascending` viene usata nella [clausola orderby](./orderby-clause.md) nelle espressioni di query per specificare l'ordinamento dal più piccolo al più grande. Dato che `ascending` è l'ordinamento predefinito non è necessario specificarlo.
