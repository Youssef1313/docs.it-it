# <a name="else-c-reference"></a>#else (Riferimenti per C#)
`#else` consente di creare una direttiva condizionale composita. In questo modo, se nessuna delle espressioni delle direttive [#if](./preprocessor-if.md) o [#elif](./preprocessor-elif.md) (facoltativa) precedenti ha restituito `true`, il compilatore valuter√† tutto il codice tra `#else` e l'espressione `#endif` successiva.  
  
## <a name="remarks"></a>Osservazioni  
 La direttiva per il preprocessore `#else` deve essere seguita da [#endif](./preprocessor-endif.md). Per un esempio di utilizzo di `#else`, vedere [#if](./preprocessor-if.md).
