# <a name="access-of-shared-member-constant-member-enum-member-or-nested-type-through-an-instance-qualifying-expression-will-not-be-evaluated"></a><span data-ttu-id="11dac-102">Accesso del membro condiviso, del membro costante, del membro di enumerazione o del tipo annidato tramite un'istanza. l'espressione di qualificazione non verrà valutata</span><span class="sxs-lookup"><span data-stu-id="11dac-102">Access of shared member, constant member, enum member or nested type through an instance; qualifying expression will not be evaluated</span></span>

<span data-ttu-id="11dac-103">Una variabile di istanza di una classe o struttura viene utilizzata per accedere a una variabile `Shared`, a una proprietà, a una routine o a un evento definito nella classe o nella struttura.</span><span class="sxs-lookup"><span data-stu-id="11dac-103">An instance variable of a class or structure is used to access a `Shared` variable, property, procedure, or event defined in that class or structure.</span></span> <span data-ttu-id="11dac-104">Questo avviso può verificarsi anche se viene usata una variabile di istanza per accedere a un membro implicitamente condiviso di una classe o di una struttura, ad esempio una costante o un'enumerazione, oppure una classe o una struttura annidata.</span><span class="sxs-lookup"><span data-stu-id="11dac-104">This warning can also occur if an instance variable is used to access an implicitly shared member of a class or structure, such as a constant or enumeration, or a nested class or structure.</span></span>

<span data-ttu-id="11dac-105">Lo scopo della condivisione di un membro consiste nel creare una sola copia di tale membro e renderla disponibile per ogni istanza della classe o della struttura in cui è dichiarata.</span><span class="sxs-lookup"><span data-stu-id="11dac-105">The purpose of sharing a member is to create only a single copy of that member and make that single copy available to every instance of the class or structure in which it is declared.</span></span> <span data-ttu-id="11dac-106">È coerente con questo scopo per accedere a un membro `Shared` tramite il nome della relativa classe o struttura, anziché tramite una variabile che include una singola istanza della classe o della struttura.</span><span class="sxs-lookup"><span data-stu-id="11dac-106">It is consistent with this purpose to access a `Shared` member through the name of its class or structure, rather than through a variable that holds an individual instance of that class or structure.</span></span>

<span data-ttu-id="11dac-107">L'accesso a un membro `Shared` tramite una variabile di istanza può rendere il codice più difficile da comprendere nascondendo il fatto che il membro è `Shared`.</span><span class="sxs-lookup"><span data-stu-id="11dac-107">Accessing a `Shared` member through an instance variable can make your code more difficult to understand by obscuring the fact that the member is `Shared`.</span></span> <span data-ttu-id="11dac-108">Inoltre, se tale accesso è parte di un'espressione che esegue altre azioni, ad esempio una routine `Function` che restituisce un'istanza del membro condiviso, Visual Basic ignora l'espressione e qualsiasi altra azione che altrimenti verrebbe eseguita.</span><span class="sxs-lookup"><span data-stu-id="11dac-108">Furthermore, if such access is part of an expression that performs other actions, such as a `Function` procedure that returns an instance of the shared member, Visual Basic bypasses the expression and any other actions it would otherwise perform.</span></span>

<span data-ttu-id="11dac-109">Per ulteriori informazioni e un esempio, vedere [Shared](../modifiers/shared.md).</span><span class="sxs-lookup"><span data-stu-id="11dac-109">For more information and an example, see [Shared](../modifiers/shared.md).</span></span>

<span data-ttu-id="11dac-110">Per impostazione predefinita, si tratta di un messaggio di avviso.</span><span class="sxs-lookup"><span data-stu-id="11dac-110">By default, this message is a warning.</span></span> <span data-ttu-id="11dac-111">Per altre informazioni su come nascondere gli avvisi o considerarli come errori, vedere [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).</span><span class="sxs-lookup"><span data-stu-id="11dac-111">For more information about hiding warnings or treating warnings as errors, see [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).</span></span>

## <a name="example"></a><span data-ttu-id="11dac-113">Esempio</span><span class="sxs-lookup"><span data-stu-id="11dac-113">Example</span></span>

<span data-ttu-id="11dac-114">L'esempio seguente genera l'BC42025:</span><span class="sxs-lookup"><span data-stu-id="11dac-114">The following example generates bc42025:</span></span>

```vb
Public Class TestClass
    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        Dim tc As New TestClass()
        tc.SayHello() ' BC42025.
    End Sub
End Module
```

## <a name="to-address-this-warning"></a><span data-ttu-id="11dac-115">Per risolvere questo avviso</span><span class="sxs-lookup"><span data-stu-id="11dac-115">To address this warning</span></span>

```vb

    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        TestClass.SayHello()
    End Sub
End Module
```
