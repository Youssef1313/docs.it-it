# <a name="error-saving-temporary-win32-resource-file-filename-error-message"></a>Errore durante il salvataggio del file di risorse Win32 temporaneo '\<filename >': \<messaggio di errore >
Il compilatore Visual Basic chiama assembly linker (al. exe, noto anche come ALink) per generare un assembly con un manifesto. Il linker ha segnalato un errore durante il tentativo di ottenere un nome di file in cui scrivere una risorsa in memoria.  
  
 **ID errore:** BC30137  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Esaminare il messaggio di errore tra virgolette e consultare l'argomento [al. exe](../../../framework/tools/al-exe-assembly-linker.md). per altre spiegazioni e consigli.  
  
2. Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
