# <a name="unable-to-link-to-resource-file-filename-error-message"></a>Impossibile eseguire il collegamento al file di risorse '\<filename >': \<messaggio di errore >
Il compilatore Visual Basic chiama assembly linker (al. exe, noto anche come ALink) per generare un assembly con un manifesto. Il linker ha segnalato un errore durante il collegamento di un file di risorse COM+ nativo dall'assembly.  
  
 **ID errore:** BC30144  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Esaminare il messaggio di errore tra virgolette e consultare l'argomento [al. exe](../../../framework/tools/al-exe-assembly-linker.md). per altre spiegazioni e consigli.  
  
2. Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
