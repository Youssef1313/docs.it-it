# <a name="error-creating-win32-resources-error-message"></a>Errore durante la creazione delle risorse Win32: \<messaggio di errore >
Il compilatore Visual Basic chiama assembly linker (al. exe, noto anche come ALink) per generare un assembly con un manifesto. Il linker ha rilevato un errore nella creazione di una risorsa in memoria. Potrebbe trattarsi di un problema di ambiente o di memoria insufficiente nel computer.  
  
 **ID errore:** BC30136  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Esaminare il messaggio di errore tra virgolette e consultare l'argomento [al. exe](../../../framework/tools/al-exe-assembly-linker.md). per altre spiegazioni e consigli.  
  
2. Se l'errore persiste, raccogliere informazioni sulla situazione contingente e informare il Servizio Supporto Tecnico Clienti Microsoft.
