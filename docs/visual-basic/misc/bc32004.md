# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>' \<name >' è dichiarato nel progetto ' \<projectname1 >', a cui non fa riferimento il progetto ' \<projectname2 >'
Un elemento di programmazione è definito in un progetto al quale il progetto corrente non fa riferimento.  
  
 **ID errore:** BC32004  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Nel progetto corrente aggiungere un riferimento al progetto che contiene l'elemento desiderato.
