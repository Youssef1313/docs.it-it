# <a name="resume-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni 'Resume' non sono valide nella finestra di controllo immediato
Le istruzioni Resume possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30714  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere le istruzioni `Resume` dal codice di debug.
