# <a name="inherits-statement-must-precede-all-declarations-in-a-class"></a>L'istruzione 'Inherits' deve precedere tutte le dichiarazioni in una classe
Le istruzioni `Inherits` devono trovarsi all'inizio delle definizioni di classe.  
  
 **ID errore:** BC30683  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Spostare l'istruzione `Inherits` all'inizio della classe.
