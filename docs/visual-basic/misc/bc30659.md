# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>Il tipo di attributo\<della proprietà o del campo ' name >' non è valido
È possibile che si stia usando un costruttore di attributo che non accetta un tipo di attributo valido, anche se l'attributo può avere altri costruttori appropriati.  
  
 **ID errore:** BC30659  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Controllare l'utilizzo di questo attributo.
