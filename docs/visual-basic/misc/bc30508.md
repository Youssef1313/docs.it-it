# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<Name >' non può esporre \<il tipo tipo1 \<> nell' \<identificatore > tipo2\<>' typeName > \<' tramite il nome >
Si è provato a esporre un tipo privato al di fuori della classe pubblica `MyClass`.  
  
 **ID errore:** BC30508  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Dichiarare il tipo `Public`.
