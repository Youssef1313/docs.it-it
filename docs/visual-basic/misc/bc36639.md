# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="08cca-102">Non è possibile usare il parametro ' ByRef '' \<parametername >' in un'espressione lambda</span><span class="sxs-lookup"><span data-stu-id="08cca-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="08cca-103">Un'espressione lambda dichiarata all'interno di `Sub` o di una funzione non può usare alcun parametro `ByRef` di quella funzione `Sub` .</span><span class="sxs-lookup"><span data-stu-id="08cca-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="08cca-104">Ad esempio, il codice seguente causerà questo errore perché l'espressione lambda usa il parametro `ByRef` `n` .</span><span class="sxs-lookup"><span data-stu-id="08cca-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb

'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="08cca-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="08cca-106">To correct this error</span></span>

    ```vb  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
