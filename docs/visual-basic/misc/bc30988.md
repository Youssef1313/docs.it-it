# <a name="type-or-with-expected"></a><span data-ttu-id="a55b3-102">È previsto il tipo o 'With'</span><span class="sxs-lookup"><span data-stu-id="a55b3-102">Type or 'With' expected</span></span>
<span data-ttu-id="a55b3-103">Quando si dichiara un'istanza di una classe, la parola chiave `New` deve essere seguita da un nome di tipo o da `With`.</span><span class="sxs-lookup"><span data-stu-id="a55b3-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="a55b3-104">Ad esempio, le istruzioni seguenti dichiarano che `client` è un'istanza della classe `Customer` .</span><span class="sxs-lookup"><span data-stu-id="a55b3-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="a55b3-105">Il nome di tipo `Customer` segue `New`.</span><span class="sxs-lookup"><span data-stu-id="a55b3-105">The type name `Customer` follows `New`.</span></span>  
  
```vb  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="a55b3-106">A partire da Visual Basic 2008, è possibile dichiarare un oggetto come istanza di un tipo anonimo, nel qual caso non si specifica un tipo di dati.</span><span class="sxs-lookup"><span data-stu-id="a55b3-106">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="a55b3-107">Nelle dichiarazioni di tipo anonimo la parola chiave `With` segue `New`.</span><span class="sxs-lookup"><span data-stu-id="a55b3-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```vb  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
