<span data-ttu-id="69444-103">Non è possibile dedurre da questi argomenti i tipi di dati dei parametri di tipo\<nel metodo di estensione ' methodName >' definito in '\<typeName >' perché sono possibili più tipi.</span><span class="sxs-lookup"><span data-stu-id="69444-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="69444-104">Per correggere l'errore, provare a specificare i tipi di dati in modo esplicito.</span><span class="sxs-lookup"><span data-stu-id="69444-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="69444-105">Si è provato a usare l'inferenza del tipo per determinare il tipo o i tipi di dati del parametro o dei parametri di tipo durante la valutazione di una chiamata a un metodo di estensione generico.</span><span class="sxs-lookup"><span data-stu-id="69444-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="69444-106">Il compilatore rileva più tipi di dati per uno o più parametri di tipo e genera questo errore.</span><span class="sxs-lookup"><span data-stu-id="69444-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="69444-107">Quando non è possibile specificare gli argomenti (ad esempio per gli operatori di query nelle espressioni di query), il messaggio di errore visualizzato non contiene la seconda frase.</span><span class="sxs-lookup"><span data-stu-id="69444-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="69444-108">Il codice seguente illustra l'errore.</span><span class="sxs-lookup"><span data-stu-id="69444-108">The following code demonstrates the error.</span></span>

```vb

Imports System.Runtime.CompilerServices
Module Module1
    Sub Main()

        Dim caller As New Class1
        '' Not valid.
        'caller.targetExtension(1, "2")

    End Sub

    <Extension()> _
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```
