# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>È in conflitto\<con ' name1 >', che viene creato in modo\<implicito per ' \<name2 >'\<nel tipo >' name3 >'
Il nome di un membro di tipo è in conflitto con un altro membro creato in modo implicito.  
  
 **ID errore:** BC31058  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rinominare il membro dichiarato in modo esplicito per rimuovere il conflitto di denominazione.
