# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<memberName >' è già dichiarato da'\<ProcedureName >', che è stato generato per questo \<ContainerName >

Un membro è stato dichiarato con lo stesso nome di un membro dichiarato dal Common Language Runtime.

**ID errore:** BC30733

## <a name="to-correct-this-error"></a>Per correggere l'errore

- Modificare il nome del membro per rimuovere il conflitto.
