# <a name="locale-id-name-is-not-supported-on-this-system"></a>L'ID delle impostazioni locali '\<Name >' non è supportato in questo sistema
Si sta tentando di usare `VbStrConv`, ma il sistema non supporta le impostazioni locali specificate.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Aggiornare il computer in modo che supporti le impostazioni locali oppure eseguire l'applicazione in un computer dotato di questo supporto.
