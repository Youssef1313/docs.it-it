# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="f9424-102">Gli operandi 'If' non possono essere argomenti denominati</span><span class="sxs-lookup"><span data-stu-id="f9424-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="f9424-103">L'uso degli argomenti denominati negli operandi dell'operatore `If` non è valido.</span><span class="sxs-lookup"><span data-stu-id="f9424-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="f9424-104">L'esempio seguente genera questo errore:</span><span class="sxs-lookup"><span data-stu-id="f9424-104">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="f9424-105">Questo comportamento è diverso da quello della funzione `IIf` , che consente argomenti denominati, come mostrato nel codice seguente:</span><span class="sxs-lookup"><span data-stu-id="f9424-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="f9424-106">**ID errore:** BC33105</span><span class="sxs-lookup"><span data-stu-id="f9424-106">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f9424-107">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="f9424-107">To correct this error</span></span>  
  
- <span data-ttu-id="f9424-108">Rimuovere le assegnazioni dagli operandi, come mostrato nel codice seguente.</span><span class="sxs-lookup"><span data-stu-id="f9424-108">Remove the name assignments from the operands, as shown in the following code.</span></span>  
  
    ```vb  
    result = If(i > 0, "positive", "not positive")  
    ```
