# <a name="extension-methods-can-be-defined-only-in-modules"></a>I metodi di estensione possono essere definiti solo nei moduli
Questo errore si verifica quando un metodo di estensione Ã¨ stato definito all'esterno di un modulo. In Visual Basic, tutti i metodi di estensione devono essere definiti all'interno di moduli standard.  
  
 **ID errore:** BC36551  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Inserire il metodo di estensione in un modulo.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente estende la classe `String` aggiungendo un metodo `Print` .  
  
```vb  
Imports StringUtility  
Imports System.Runtime.CompilerServices  
Namespace StringUtility  
    <Extension()> _  
    Module StringExtensions  
        <Extension()> _  
        Public Sub Print (ByVal str As String)  
            Console.WriteLine(str)  
        End Sub  
    End Module  
End Namespace  
```
