## <a name="global-namespaces-and-embedded-expressions"></a><span data-ttu-id="dd691-120">Spazi dei nomi globali ed espressioni incorporate</span><span class="sxs-lookup"><span data-stu-id="dd691-120">Global Namespaces and Embedded Expressions</span></span>
