 <span data-ttu-id="6811c-106">Come concetto di base, LINQ to Objects rappresenta un nuovo approccio alle raccolte.</span><span class="sxs-lookup"><span data-stu-id="6811c-106">In a basic sense, LINQ to Objects represents a new approach to collections.</span></span> <span data-ttu-id="6811c-107">In passato, era necessario scrivere cicli `For Each` complessi che specificavano come recuperare i dati da una raccolta.</span><span class="sxs-lookup"><span data-stu-id="6811c-107">In the old way, you had to write complex `For Each` loops that specified how to retrieve data from a collection.</span></span> <span data-ttu-id="6811c-108">Con l'approccio LINQ, è possibile scrivere il codice dichiarativo che descrive i dati da recuperare.</span><span class="sxs-lookup"><span data-stu-id="6811c-108">In the LINQ approach, you write declarative code that describes what you want to retrieve.</span></span>  
  
 <span data-ttu-id="6811c-109">Le query LINQ offrono anche tre vantaggi principali rispetto ai cicli `For Each` tradizionali:</span><span class="sxs-lookup"><span data-stu-id="6811c-109">In addition, LINQ queries offer three main advantages over traditional `For Each` loops:</span></span>  
  
1. <span data-ttu-id="6811c-110">Sono più brevi e leggibili, soprattutto quando si filtrano più condizioni.</span><span class="sxs-lookup"><span data-stu-id="6811c-110">They are more concise and readable, especially when filtering multiple conditions.</span></span>

3. <span data-ttu-id="6811c-112">Possono essere trasferiti in altre origini dati con modifiche minime o nulle.</span><span class="sxs-lookup"><span data-stu-id="6811c-112">They can be ported to other data sources with little or no modification.</span></span>  
  
 <span data-ttu-id="6811c-113">In generale, più è complessa l'operazione da eseguire sui dati, maggiore sarà il vantaggio che si potrà trarre dall'uso di LINQ rispetto alle tecniche di iterazione tradizionali.</span><span class="sxs-lookup"><span data-stu-id="6811c-113">In general, the more complex the operation you want to perform on the data, the more benefit you will realize by using LINQ instead of traditional iteration techniques.</span></span>
