# <a name="elementary-data-types-visual-basic"></a>Tipi di dati elementari (Visual Basic)
Visual Basic fornisce un set di tipi di dati predefiniti che è possibile usare per molti degli elementi di programmazione. Questa sezione descrive questi tipi e come usarli.  
  
> [!NOTE]
> Ogni tipo di dati elementare in Visual Basic è supportato da una struttura o da una classe presente nello <xref:System> spazio dei nomi. Il compilatore USA ogni parola chiave del tipo di dati come alias per la struttura o la classe sottostante. Ad esempio, la dichiarazione di una variabile tramite la parola `Byte` riservata equivale alla dichiarazione usando il nome <xref:System.Byte?displayProperty=nameWithType>completo della struttura.
