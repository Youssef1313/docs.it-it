# <a name="entity-key"></a><span data-ttu-id="237d5-102">chiave di entità</span><span class="sxs-lookup"><span data-stu-id="237d5-102">entity key</span></span>
<span data-ttu-id="237d5-103">Una *chiave di entità* è una [Proprietà](property.md) o un set di proprietà di un [tipo di entità](entity-type.md) usate per determinare l'identità.</span><span class="sxs-lookup"><span data-stu-id="237d5-103">An *entity key* is a [property](property.md) or a set of properties of an [entity type](entity-type.md) that are used to determine identity.</span></span> <span data-ttu-id="237d5-104">Le proprietà che costituiscono una chiave di entità vengono scelte in fase di progettazione.</span><span class="sxs-lookup"><span data-stu-id="237d5-104">The properties that make up an entity key are chosen at design time.</span></span> <span data-ttu-id="237d5-105">I valori delle proprietà della chiave di entità devono identificare in modo univoco un'istanza del tipo di entità all'interno di un [set di entità](entity-set.md) in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="237d5-105">The values of entity key properties must uniquely identify an entity type instance within an [entity set](entity-set.md) at run time.</span></span> <span data-ttu-id="237d5-106">Le proprietà che costituiscono una chiave di entità devono essere scelte per garantire univocità delle istanze in un set di entità.</span><span class="sxs-lookup"><span data-stu-id="237d5-106">The properties that make up an entity key should be chosen to guarantee uniqueness of instances in an entity set.</span></span>  
  
 <span data-ttu-id="237d5-107">Di seguito sono elencati i requisiti che consentono a un set di proprietà di essere una chiave di entità:</span><span class="sxs-lookup"><span data-stu-id="237d5-107">The following are the requirements for a set of properties to be an entity key:</span></span>  
  
- <span data-ttu-id="237d5-108">Non possono esistere due chiavi di entità identiche all'interno di un set di entità,</span><span class="sxs-lookup"><span data-stu-id="237d5-108">No two entity keys within an entity set can be identical.</span></span> <span data-ttu-id="237d5-109">vale a dire che, per due entità qualsiasi all'interno di un set di entità, i valori per tutte le proprietà che costituiscono una chiave non possono essere gli stessi.</span><span class="sxs-lookup"><span data-stu-id="237d5-109">That is, for any two entities within an entity set, the values for all of the properties that constitute a key cannot be the same.</span></span> <span data-ttu-id="237d5-110">Tuttavia, alcuni valori (ma non tutti) che costituiscono una chiave di entità possono essere gli stessi.</span><span class="sxs-lookup"><span data-stu-id="237d5-110">However, some (but not all) of the values that make up an entity key can be the same.</span></span>  
  
- <span data-ttu-id="237d5-111">Una chiave di entità deve essere costituita da un set di [proprietà di tipo primitivo](entity-data-model-primitive-data-types.md)non nullable, non modificabile.</span><span class="sxs-lookup"><span data-stu-id="237d5-111">An entity key must consist of a set of non-nullable, immutable, [primitive type properties](entity-data-model-primitive-data-types.md).</span></span>  
  
- <span data-ttu-id="237d5-112">Le proprietà che costituiscono una chiave di entità per un determinato tipo di entità non possono essere modificate.</span><span class="sxs-lookup"><span data-stu-id="237d5-112">The properties that make up an entity key for a given entity type cannot change.</span></span> <span data-ttu-id="237d5-113">Non è possibile consentire più di una possibile chiave di entità per un determinato tipo di entità. Le chiavi surrogate non sono supportate.</span><span class="sxs-lookup"><span data-stu-id="237d5-113">You cannot allow more than one possible entity key for a given entity type; surrogate keys are not supported.</span></span>  
  
- <span data-ttu-id="237d5-114">Quando un'entità è coinvolta in una gerarchia di ereditarietà, l'entità radice deve contenere tutte le proprietà che costituiscono la chiave di entità e la chiave di entità deve essere definita sul tipo di entità radice.</span><span class="sxs-lookup"><span data-stu-id="237d5-114">When an entity is involved in an inheritance hierarchy, the root entity must contain all the properties that make up the entity key, and the entity key must be defined on the root entity type.</span></span> <span data-ttu-id="237d5-115">Per ulteriori informazioni, vedere [Entity Data Model: ereditarietà](entity-data-model-inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="237d5-115">For more information, see [Entity Data Model: Inheritance](entity-data-model-inheritance.md).</span></span>
