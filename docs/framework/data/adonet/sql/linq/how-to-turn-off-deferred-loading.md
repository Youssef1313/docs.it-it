# <a name="how-to-turn-off-deferred-loading"></a>Procedura: Disattivare il caricamento posticipato
È possibile disattivare il caricamento posticipato impostando <xref:System.Data.Linq.DataContext.DeferredLoadingEnabled%2A> su `false`. Per ulteriori informazioni, vedere il [caricamento posticipato rispetto al caricamento immediato](../../../../../../docs/framework/data/adonet/sql/linq/deferred-versus-immediate-loading.md).  
  
> [!NOTE]
> Il caricamento posticipato viene disattivato implicitamente quando è disattivato il rilevamento degli oggetti. Per altre informazioni, vedere [Procedura: Recuperare informazioni in sola lettura](../../../../../../docs/framework/data/adonet/sql/linq/how-to-retrieve-information-as-read-only.md).
