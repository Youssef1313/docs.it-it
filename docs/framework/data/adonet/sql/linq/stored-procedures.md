# <a name="stored-procedures"></a>Stored procedure
[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]utilizza i metodi nel modello a oggetti per rappresentare le stored procedure nel database. Per definire metodi come stored procedure, applicare l'attributo <xref:System.Data.Linq.Mapping.FunctionAttribute> quindi, dove richiesto, l'attributo <xref:System.Data.Linq.Mapping.ParameterAttribute>. Per ulteriori informazioni, vedere [il modello a oggetti LINQ to SQL](the-linq-to-sql-object-model.md).  
  
 Gli sviluppatori che usano Visual Studio utilizzeranno in genere il Object Relational Designer per eseguire il mapping delle stored procedure. Negli argomenti elencati in questa sezione viene illustrato come formare e chiamare questi metodi nell'applicazione quando si scrive codice personalizzato.
