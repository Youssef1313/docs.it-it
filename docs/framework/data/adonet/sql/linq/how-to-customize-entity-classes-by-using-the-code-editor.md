# <a name="how-to-customize-entity-classes-by-using-the-code-editor"></a>Procedura: Personalizzare classi di entità mediante l'editor del codice
Gli sviluppatori che usano Visual Studio possono usare la Object Relational Designer per creare o personalizzare le classi di entità.  
  
 È anche possibile usare l'editor di codice di Visual Studio per scrivere il proprio codice di mapping o per personalizzare il codice che è già stato generato. Per altre informazioni, vedere [mapping basato su attributi](attribute-based-mapping.md).  
  
 Negli argomenti di questa sezione viene descritto come personalizzare il modello a oggetti.  
  
 [Procedura: Specificare i nomi dei database](how-to-specify-database-names.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.DatabaseAttribute.Name%2A>.  
  
 [Procedura: Rappresenta tabelle come classi](how-to-represent-tables-as-classes.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.TableAttribute>.  
  
 [Procedura: Rappresenta colonne come membri di classe](how-to-represent-columns-as-class-members.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.ColumnAttribute>.  
  
 [Procedura: Rappresenta chiavi primarie](how-to-represent-primary-keys.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.ColumnAttribute.IsPrimaryKey%2A>.  
  
 [Procedura: Mapping di relazioni di database](how-to-map-database-relationships.md)  
 Vengono forniti esempi relativi all'utilizzo dell'attributo <xref:System.Data.Linq.Mapping.AssociationAttribute>.  
  
 [Procedura: Rappresenta colonne come generate dal database](how-to-represent-columns-as-database-generated.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.ColumnAttribute.IsDbGenerated%2A>.  
  
 [Procedura: Rappresenta colonne come timestamp o versione](how-to-represent-columns-as-timestamp-or-version-columns.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.ColumnAttribute.IsVersion%2A>.  
  
 [Procedura: Specificare i tipi di dati del database](how-to-specify-database-data-types.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.ColumnAttribute.DbType%2A>.  
  
 [Procedura: Rappresenta colonne calcolate](how-to-represent-computed-columns.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.ColumnAttribute.Expression%2A>.  
  
 [Procedura: Specificare i campi di archiviazione privata](how-to-specify-private-storage-fields.md)

 [Procedura: Rappresenta colonne che consentono valori null](how-to-represent-columns-as-allowing-null-values.md)  
 Viene descritto come usare <xref:System.Data.Linq.Mapping.ColumnAttribute.CanBeNull%2A>.  
  
 [Procedura: Mapping di gerarchie di ereditarietà](how-to-map-inheritance-hierarchies.md)  
 Vengono descritti i mapping necessari per specificare una gerarchia di ereditarietà.  
  
 [Procedura: Impostazione del controllo dei conflitti di concorrenza](how-to-specify-concurrency-conflict-checking.md)
