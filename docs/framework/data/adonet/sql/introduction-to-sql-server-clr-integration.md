 <span data-ttu-id="c2f5b-106">Con CLR incluso in Microsoft SQL Server (integrazione di CLR), è possibile creare nuove stored procedure, trigger, funzioni definite dall'utente e aggregati definiti dall'utente nel codice gestito.</span><span class="sxs-lookup"><span data-stu-id="c2f5b-106">With the CLR hosted in Microsoft SQL Server (called CLR integration), you can author stored procedures, triggers, user-defined functions, user-defined types, and user-defined aggregates in managed code.</span></span> <span data-ttu-id="c2f5b-107">Poiché il codice gestito viene processato con il codice nativo prima dell'esecuzione, in determinati scenari è possibile ottenere significativi aumenti di prestazioni.</span><span class="sxs-lookup"><span data-stu-id="c2f5b-107">Because managed code compiles to native code prior to execution, you can achieve significant performance increases in some scenarios.</span></span>
