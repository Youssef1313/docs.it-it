# <a name="xml-data-in-sql-server"></a>Dati XML in SQL Server
SQL Server espone le funzionalit√† di SQLXML all'interno di .NET Framework. Gli sviluppatori possono scrivere applicazioni che accedono ai dati XML da un'istanza SQL Server, trasportano i dati nell'ambiente .NET Framework, elaborano i dati e inviano gli aggiornamenti a SQL Server. I dati XML possono essere usati in vari modi in SQL Server, ad esempio nell'archiviazione dei dati e come valori di parametro per il recupero dei dati. La classe **SQLXML** nel .NET Framework fornisce il supporto lato client per l'utilizzo di dati archiviati in una colonna XML all'interno SQL Server. Per altre informazioni, vedere "SQLXML Managed Classes" nella Documentazione online di SQL Server.
