### <a name="graphics-rendering-tiers"></a><span data-ttu-id="fa4c7-118">Livelli di rendering della grafica</span><span class="sxs-lookup"><span data-stu-id="fa4c7-118">Graphics Rendering Tiers</span></span>

 <span data-ttu-id="fa4c7-121">A tale scopo, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] fornisce la funzionalità per determinare la funzionalità grafica di un sistema in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-121">To achieve this, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] provides functionality to determine the graphics capability of a system at runtime.</span></span> <span data-ttu-id="fa4c7-122">La funzionalità grafica è determinata dalla categorizzazione della scheda video come uno dei tre livelli di funzionalità di rendering.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-122">Graphics capability is determined by categorizing the video card as one of three rendering capability tiers.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]<span data-ttu-id="fa4c7-123">espone un'API che consente a un'applicazione di eseguire query sul livello di funzionalità di rendering.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-123">exposes an API that allows an application to query the rendering capability tier.</span></span> <span data-ttu-id="fa4c7-124">L'applicazione può quindi usare percorsi di codice diversi in fase di esecuzione a seconda del livello di rendering supportato dall'hardware.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-124">Your application can then take different code paths at run time depending on the rendering tier supported by the hardware.</span></span>  
  
 <span data-ttu-id="fa4c7-125">Le funzionalità dell'hardware grafico che hanno effetto sui livelli di rendering sono:</span><span class="sxs-lookup"><span data-stu-id="fa4c7-125">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>

- <span data-ttu-id="fa4c7-135">**Livello di rendering 0** Nessuna accelerazione hardware grafico.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-135">**Rendering Tier 0** No graphics hardware acceleration.</span></span> <span data-ttu-id="fa4c7-136">Il livello della versione di DirectX è inferiore alla versione 7,0.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-136">The DirectX version level is less than version 7.0.</span></span>  
  
- <span data-ttu-id="fa4c7-137">**Livello di rendering 1** Accelerazione hardware grafica parziale.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="fa4c7-138">Il livello della versione di DirectX è maggiore o uguale alla versione 7,0 e **inferiore** alla versione 9,0.</span><span class="sxs-lookup"><span data-stu-id="fa4c7-138">The DirectX version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>
