<span data-ttu-id="7532e-103">Rinvia un valore di proprietà a un valore associato a dati, creando un oggetto espressione intermedia e interpretando il contesto dei dati che si applica all'elemento e la relativa associazione in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="7532e-103">Defers a property value to be a data-bound value, creating an intermediate expression object and interpreting the data context that applies to the element and its binding at run time.</span></span>  
  
## <a name="binding-expression-usage"></a><span data-ttu-id="7532e-104">Utilizzo delle espressioni di binding</span><span class="sxs-lookup"><span data-stu-id="7532e-104">Binding Expression Usage</span></span>  
  
```xaml  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
