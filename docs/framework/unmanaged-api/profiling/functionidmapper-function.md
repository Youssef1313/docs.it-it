# <a name="functionidmapper-function"></a><span data-ttu-id="9cb2e-102">Funzione FunctionIDMapper</span><span class="sxs-lookup"><span data-stu-id="9cb2e-102">FunctionIDMapper Function</span></span>
<span data-ttu-id="9cb2e-103">Notifica al profiler che l'identificatore specificato di una funzione può essere rimappato a un ID alternativo da utilizzare per il [FunctionEnter2](../../../../docs/framework/unmanaged-api/profiling/functionenter2-function.md), [FunctionLeave2](../../../../docs/framework/unmanaged-api/profiling/functionleave2-function.md), e [FunctionTailcall2](../../../../docs/framework/unmanaged-api/profiling/functiontailcall2-function.md) richiamate per tale funzione.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-103">Notifies the profiler that the given identifier of a function may be remapped to an alternative ID to be used in the [FunctionEnter2](../../../../docs/framework/unmanaged-api/profiling/functionenter2-function.md), [FunctionLeave2](../../../../docs/framework/unmanaged-api/profiling/functionleave2-function.md), and [FunctionTailcall2](../../../../docs/framework/unmanaged-api/profiling/functiontailcall2-function.md) callbacks for that function.</span></span> <span data-ttu-id="9cb2e-104">`FunctionIDMapper` consente inoltre al profiler di indicare se vuole ricevere i callback per tale funzione.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-104">`FunctionIDMapper` also enables the profiler to indicate whether it wants to receive callbacks for that function.</span></span>

 <span data-ttu-id="9cb2e-114">Il `FunctionIDMapper` funzione è un callback.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-114">The `FunctionIDMapper` function is a callback.</span></span> <span data-ttu-id="9cb2e-115">Viene implementato dal profiler per modificare il mapping di un ID di funzione a un altro identificatore che è più utile per il profiler.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-115">It is implemented by the profiler to remap a function ID to some other identifier that is more useful for the profiler.</span></span> <span data-ttu-id="9cb2e-116">Il `FunctionIDMapper` restituisce l'ID alternativo da utilizzare per qualsiasi funzione specificata.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-116">The `FunctionIDMapper` returns the alternate ID to be used for any given function.</span></span> <span data-ttu-id="9cb2e-117">Il motore di esecuzione quindi rispetta richiesta del profiler, passando l'ID alternativo, oltre all'ID funzione tradizionali, al profiler, vedere la `clientData` parametro il `FunctionEnter2`, `FunctionLeave2`, e `FunctionTailcall2` hook, per identificare funzione per il quale viene chiamata l'hook.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-117">The execution engine then honors the profiler's request by passing this alternate ID, in addition to the traditional function ID, back to the profiler in the `clientData` parameter of the `FunctionEnter2`, `FunctionLeave2`, and `FunctionTailcall2` hooks, to identify the function for which the hook is being called.</span></span>  
  
 <span data-ttu-id="9cb2e-118">È possibile usare la [ICorProfilerInfo:: SetFunctionIDMapper](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-setfunctionidmapper-method.md) specificare l'implementazione del metodo di `FunctionIDMapper` (funzione).</span><span class="sxs-lookup"><span data-stu-id="9cb2e-118">You can use the [ICorProfilerInfo::SetFunctionIDMapper](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-setfunctionidmapper-method.md) method to specify the implementation of the `FunctionIDMapper` function.</span></span> <span data-ttu-id="9cb2e-119">È possibile chiamare il `ICorProfilerInfo::SetFunctionIDMapper` metodo di una sola volta e si consiglia di eseguire questa operazione nel [ICorProfilerCallback:: Initialize](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-initialize-method.md) callback.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-119">You can call the `ICorProfilerInfo::SetFunctionIDMapper` method only once, and we recommend that you do so in the [ICorProfilerCallback::Initialize](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-initialize-method.md) callback.</span></span>  
  
 <span data-ttu-id="9cb2e-120">Per impostazione predefinita, si presuppone che un profiler che imposta il flag COR_PRF_MONITOR_ENTERLEAVE usando [ICorProfilerInfo:: SetEventMask](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-seteventmask-method.md), e gli hook tramite [ICorProfilerInfo::SetEnterLeaveFunctionHooks](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-setenterleavefunctionhooks-method.md) oppure [ICorProfilerInfo2::SetEnterLeaveFunctionHooks2](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo2-setenterleavefunctionhooks2-method.md), dovrebbe ricevere la `FunctionEnter2`, `FunctionLeave2`, e `FunctionTailcall2` callback per ogni funzione.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-120">By default, it is assumed that a profiler that sets the COR_PRF_MONITOR_ENTERLEAVE flag by using [ICorProfilerInfo::SetEventMask](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-seteventmask-method.md), and which sets hooks via [ICorProfilerInfo::SetEnterLeaveFunctionHooks](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo-setenterleavefunctionhooks-method.md) or [ICorProfilerInfo2::SetEnterLeaveFunctionHooks2](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo2-setenterleavefunctionhooks2-method.md), should receive the `FunctionEnter2`, `FunctionLeave2`, and `FunctionTailcall2` callbacks for every function.</span></span> <span data-ttu-id="9cb2e-121">Tuttavia, i profiler possono implementare `FunctionIDMapper` per evitare in modo selettivo riceve questi callback per determinate funzioni impostando `pbHookFunction` a `false`.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-121">However, profilers may implement `FunctionIDMapper` to selectively avoid receiving these callbacks for certain functions by setting `pbHookFunction` to `false`.</span></span>  
  
 <span data-ttu-id="9cb2e-122">I profiler devono poter supportare i casi in cui più thread di un'applicazione profilata chiamano il metodo/funzione stessa contemporaneamente.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-122">Profilers should be tolerant of cases where multiple threads of a profiled application are calling the same method/function simultaneously.</span></span> <span data-ttu-id="9cb2e-123">In questi casi, il profiler può ricevere più `FunctionIDMapper` callback per lo stesso `FunctionID`.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-123">In such cases, the profiler may receive multiple `FunctionIDMapper` callbacks for the same `FunctionID`.</span></span> <span data-ttu-id="9cb2e-124">Il profiler deve essere determinato da restituire gli stessi valori da questo callback quando viene chiamata più volte con lo stesso `FunctionID`.</span><span class="sxs-lookup"><span data-stu-id="9cb2e-124">The profiler should be certain to return the same values from this callback when it is called multiple times with the same `FunctionID`.</span></span>
