# <a name="icorprofilerobjectenumreset-method"></a>Metodo ICorProfilerObjectEnum::Reset
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT Reset ();  
```
