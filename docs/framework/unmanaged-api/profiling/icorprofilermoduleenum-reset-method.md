# <a name="icorprofilermoduleenumreset-method"></a>Metodo ICorProfilerModuleEnum::Reset
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT Reset();  
```
