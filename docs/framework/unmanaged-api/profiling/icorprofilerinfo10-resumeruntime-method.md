# <a name="icorprofilerinfo10resumeruntime-method"></a>Metodo ICorProfilerInfo10:: ResumeRuntime
  
Riprende il runtime senza eseguire un catalogo globale.   
  
## <a name="syntax"></a>Sintassi  
  
```cpp
HRESULT ResumeRuntime();
```
