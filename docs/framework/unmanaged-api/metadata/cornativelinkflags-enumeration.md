# <a name="cornativelinkflags-enumeration"></a>Enumerazione CorNativeLinkFlags
Fornisce valori di flag usati dal linker durante il collegamento del codice nativo.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
typedef enum  
{  
    nlfNone         = 0x00,  
    nlfLastError    = 0x01,  
    nlfNoMangle     = 0x02,  
    nlfMaxValue     = 0x03  
} CorNativeLinkFlags;  
```  
  
## <a name="members"></a>Members  
  
|Member|Descrizione|  
|------------|-----------------|  
|`nlfNone`|Indicates no flags.|  
|`nlfLastError`|Indicates a `setLastError` keyword.|  
|`nlfNoMangle`|Indicates a `nomangle` keyword.|  
|`nlfMaxValue`|Non usato.|
