# <a name="cvstruct-structure"></a>Struttura CVStruct
Contiene informazioni usate durante l'installazione di un modulo o un'immagine composita.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```  
  
## <a name="members"></a>Membri  
  
|Membro|Descrizione|  
|------------|-----------------|  
|Principale|Numero di build della versione principale.|  
|Secondaria|Numero di build della versione secondaria.|  
|risposta|Numero di sottocompilazione.|  
|Compila|Numero di build.|
