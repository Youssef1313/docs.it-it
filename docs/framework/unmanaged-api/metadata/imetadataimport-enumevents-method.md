# <a name="imetadataimportenumevents-method"></a>Metodo IMetaDataImport::EnumEvents
Enumera i token di definizione di evento per il token TypeDef specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT EnumEvents (   
   [in, out] HCORENUM    *phEnum,   
   [in]      mdTypeDef   td,   
   [out]     mdEvent     rEvents[],   
   [in]      ULONG       cMax,  
   [out]    ULONG        *pcEvents  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `phEnum`  
 [in, out] Puntatore all'enumeratore.  
  
 `td`  
 in Token TypeDef le cui definizioni di evento devono essere enumerate.  
  
 `rEvents`  
 out Matrice degli eventi restituiti.  
  
 `cMax`  
 [in] Dimensione massima della matrice `rEvents`.  
  
 `pcEvents`  
 out Numero effettivo di eventi restituiti in `rEvents`.  
  
## <a name="return-value"></a>Valore restituito  
  
|HRESULT|Descrizione|  
|-------------|-----------------|  
|`S_OK`|`EnumEvents` ha restituito un esito positivo.|  
|`S_FALSE`|Nessun evento da enumerare. In tal caso, `pcEvents` Ã¨ zero.|
