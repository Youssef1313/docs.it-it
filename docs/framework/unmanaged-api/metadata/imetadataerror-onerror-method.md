# <a name="imetadataerroronerror-method"></a>Metodo IMetaDataError::OnError
Fornisce la notifica degli errori che si verificano durante l'Unione dei metadati.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT OnError (  
    [in] HRESULT   hrError,   
    [in] mdToken   token  
);  
```
