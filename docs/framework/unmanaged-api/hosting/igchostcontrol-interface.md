# <a name="igchostcontrol-interface"></a>Interfaccia IGCHostControl
Fornisce un metodo che consente all'Garbage Collector di richiedere all'host di modificare i limiti della memoria virtuale.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo RequestVirtualMemLimit](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|Richiede all'host di modificare i limiti della memoria virtuale.|
