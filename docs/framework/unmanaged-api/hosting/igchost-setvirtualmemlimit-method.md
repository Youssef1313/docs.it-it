# <a name="igchostsetvirtualmemlimit-method"></a>Metodo IGCHost::SetVirtualMemLimit
Imposta la dimensione massima della memoria virtuale del runtime.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT SetVirtualMemLimit (  
    [in] SIZE_T sztMaxVirtualMemMB  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `sztMaxVirtualMemMB`  
 in Dimensione massima, in megabyte, della memoria virtuale del runtime.
