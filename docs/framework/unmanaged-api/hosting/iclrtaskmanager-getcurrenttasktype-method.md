# <a name="iclrtaskmanagergetcurrenttasktype-method"></a>Metodo ICLRTaskManager::GetCurrentTaskType
Ottiene il tipo dell'attività attualmente in esecuzione.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetCurrentTaskType(  
    [out] ETaskType *pTaskType  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `pTaskType`  
 out Puntatore a un valore dell'enumerazione [ETaskType](../../../../docs/framework/unmanaged-api/hosting/etasktype-enumeration.md) che indica il tipo di attività attualmente in esecuzione.
