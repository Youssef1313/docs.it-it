# <a name="cor_gc_thread_stats_types-enumeration"></a>Enumerazione COR_GC_THREAD_STATS_TYPES

```cpp  
typedef enum {  
    COR_GC_THREAD_HAS_PROMOTED_BYTES  = 0x00000001  
} COR_GC_THREAD_STATS_TYPES;
```  
