# <a name="icorpublishenumreset-method"></a>Metodo ICorPublishEnum::Reset
Sposta il cursore all'inizio dell'enumerazione.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT Reset ();  
```  
  
## <a name="parameters"></a>Parametri  
 `Reset` non accetta parametri.
