# <a name="icordebugmodulegetbaseaddress-method"></a>Metodo ICorDebugModule::GetBaseAddress
Ottiene l'indirizzo di base del modulo.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetBaseAddress(  
    [out] CORDB_ADDRESS *pAddress  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `pAddress`  
 out `CORDB_ADDRESS` che specifica l'indirizzo di base del modulo.  
  
## <a name="remarks"></a>Note  
 Se il modulo è un'immagine nativa, ovvero se il modulo è stato prodotto dal generatore di immagini native, NGen. exe, il relativo indirizzo di base sarà zero.
