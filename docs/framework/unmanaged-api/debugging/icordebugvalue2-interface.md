# <a name="icordebugvalue2-interface"></a>Interfaccia ICorDebugValue2
Estende l'interfaccia "ICorDebugValue" per fornire supporto per gli oggetti "ICorDebugType".  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo GetExactType](../../../../docs/framework/unmanaged-api/debugging/icordebugvalue2-getexacttype-method.md)|Ottiene un puntatore a interfaccia a un oggetto `ICorDebugType` che rappresenta l'<xref:System.Type> di questo valore.|
