# <a name="icordebugfunctionbreakpointgetfunction-method"></a>Metodo ICorDebugFunctionBreakpoint::GetFunction
Ottiene un puntatore a interfaccia a un ICorDebugFunction che fa riferimento alla funzione in cui è impostato il punto di interruzione.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetFunction (  
    [out] ICorDebugFunction  **ppFunction  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `ppFunction`  
 out Puntatore all'indirizzo della funzione in cui è impostato il punto di interruzione.
