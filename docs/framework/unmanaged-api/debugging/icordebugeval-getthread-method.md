# <a name="icordebugevalgetthread-method"></a>Metodo ICorDebugEval::GetThread
Ottiene il thread in cui è in esecuzione la valutazione o verrà eseguito.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetThread (  
    [out] ICorDebugThread   **ppThread  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `ppThread`  
 out Puntatore all'indirizzo di un oggetto ICorDebugThread che rappresenta il thread.
