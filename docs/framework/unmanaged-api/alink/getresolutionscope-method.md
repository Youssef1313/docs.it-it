# <a name="getresolutionscope-method"></a>Metodo GetResolutionScope
Recupera l'ambito di un tipo specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetResolutionScope(  
    mdAssembly  AssemblyID,  
    mdToken     FileToken,  
    mdToken     TargetFile,  
    mdToken*    pScope  
) PURE;  
```  
  
## <a name="parameters"></a>Parametri  
 `AssemblyID`  
 ID dell'assembly.  
  
 `FileToken`  
 File che necessita di un riferimento.  
  
 `TargetFile`  
 Token del file in cui Ã¨ definito il tipo, generalmente recuperato con il [metodo ImportFile](importfile-method.md).  
  
 `pScope`

## <a name="return-value"></a>Valore restituito  
 Restituisce S_OK se il metodo ha esito positivo.

 Richiede ALink. h.
