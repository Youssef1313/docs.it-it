# <a name="fusion-structures"></a>Strutture Fusion
Questa sezione descrive le strutture non gestite utilizzate dall'API Fusion.  
  
## <a name="in-this-section"></a>In questa sezione  
 [Struttura ASSEMBLY_INFO](assembly-info-structure.md)  
 Contiene informazioni su un assembly registrato nel Global Assembly Cache.  
  
 [Struttura FUSION_INSTALL_REFERENCE](fusion-install-reference-structure.md)  
 Rappresenta un riferimento che un'applicazione esegue a un assembly installato dall'applicazione nell'Global Assembly Cache.  
  
 [Struttura IDENTITY_ATTRIBUTE](identity-attribute-structure.md)  
 Contiene informazioni sugli attributi di metadati relativi a un'istanza di [IDefinitionIdentity](idefinitionidentity-interface.md) .  
  
 [Struttura IDENTITY_ATTRIBUTE_BLOB](identity-attribute-blob-structure.md)  
 Contiene le informazioni sull'offset per una struttura [IDENTITY_ATTRIBUTE](identity-attribute-structure.md) associata.
