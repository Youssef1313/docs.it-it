# <a name="iassemblyenumclone-method"></a>Metodo IAssemblyEnum::Clone
Crea una copia superficiale di questo oggetto [IAssemblyEnum](iassemblyenum-interface.md) .  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyEnum   **ppEnum  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `ppEnum`  
 out Puntatore alla copia.
