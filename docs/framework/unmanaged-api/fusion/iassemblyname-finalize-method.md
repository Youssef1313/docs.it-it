# <a name="iassemblynamefinalize-method"></a>Metodo IAssemblyName::Finalize
Consente a questo oggetto [IAssemblyName](iassemblyname-interface.md) di rilasciare risorse ed eseguire altre operazioni di pulizia prima che venga chiamato il relativo distruttore.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT Finalize ();  
```
