## <a name="remarks"></a><span data-ttu-id="8dd46-178">Note</span><span class="sxs-lookup"><span data-stu-id="8dd46-178">Remarks</span></span>  
 <span data-ttu-id="8dd46-179">L'elemento `<Assembly>` definisce i criteri di runtime per tutti i tipi in un assembly.</span><span class="sxs-lookup"><span data-stu-id="8dd46-179">The `<Assembly>` element defines runtime policy for all the types in an assembly.</span></span> <span data-ttu-id="8dd46-180">È diverso dall'elemento [\<Library>](library-element-net-native.md), che specifica una libreria, ma dipende dagli elementi figlio per definire i criteri di reflection di runtime.</span><span class="sxs-lookup"><span data-stu-id="8dd46-180">It differs from the [\<Library>](library-element-net-native.md) element, which specifies a library but depends on its child elements to define runtime reflection policy.</span></span> <span data-ttu-id="8dd46-181">L'elemento `<Assembly>` si applica a tutti i tipi in un assembly a meno che questi non siano sottoposti a override da un elemento figlio.</span><span class="sxs-lookup"><span data-stu-id="8dd46-181">The `<Assembly>` element applies to all the types in an assembly unless they are overridden by a child element.</span></span>  
  
 <span data-ttu-id="8dd46-182">L'esempio seguente illustra come applicare i criteri di runtime a tutti i tipi presenti negli assembly all'interno del pacchetto di app assegnando un valore "\*Application\*" all'attributo `Name`.</span><span class="sxs-lookup"><span data-stu-id="8dd46-182">The following example shows how you can apply runtime policy to all the types in assemblies within your app package by assigning the `Name` attribute a value of "\*Application\*".</span></span> <span data-ttu-id="8dd46-183">L'elemento `<Assembly>` deve essere figlio dell'elemento [\<Application>](application-element-net-native.md).</span><span class="sxs-lookup"><span data-stu-id="8dd46-183">The `<Assembly>` element must be a child of the [\<Application>](application-element-net-native.md) element.</span></span>
