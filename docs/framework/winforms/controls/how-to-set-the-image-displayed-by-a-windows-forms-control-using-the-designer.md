# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>Procedura: Imposta l'immagine visualizzata da un controllo Windows Forms usando la finestra di progettazione

Diversi controlli Windows Forms possono visualizzare immagini. L'immagine può essere un'icona che chiarisce lo scopo del controllo, ad esempio un'icona del disco su un pulsante che indica il comando **Salva** . In alternativa, l'icona può essere un'immagine di sfondo per dare al controllo l'aspetto desiderato.

## <a name="to-set-the-image-displayed-by-a-control"></a>Per impostare l'immagine visualizzata da un controllo

1. Nella finestra **Proprietà** selezionare la proprietà **Image** o **BackgroundImage** del controllo, quindi fare clic sui puntini di sospensione (![il pulsante con i puntini](./media/visual-studio-ellipsis-button.png)di sospensione in Visual Studio) per visualizzare la **risorsa Select** . finestra di dialogo.

2. Selezionare l'immagine che si desidera visualizzare.
