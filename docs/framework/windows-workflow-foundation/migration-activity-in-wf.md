# <a name="migration-activity-in-wf"></a>Attività Migration in WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]fornisce l' <xref:System.Activities.Statements.Interop> attività per l'esecuzione di attività che derivano dall'attività all'interno di un flusso <xref:System.Activities.Activity>di lavoro basato su. Per ulteriori informazioni, vedere la sezione relativa alle [linee guida](migration-guidance.md) per la migrazione.  
  
> [!NOTE]
> L' <xref:System.Activities.Statements.Interop> attività non viene visualizzata nella casella degli strumenti di progettazione flussi di lavoro a meno che l'impostazione del **Framework di destinazione** del progetto del flusso di lavoro non sia impostata su **.NET Framework 4** o versione successiva.
