# <a name="tlsstreamm_worker-field"></a>Campo TlsStream. m_Worker

Rappresenta lo stato del flusso SSL.

## <a name="syntax"></a>Sintassi

```csharp
private SslState m_Worker;
```

## <a name="field-value"></a>Valore del campo

`System.Net.Security.SslState`  
Stato del flusso SSL.

## <a name="remarks"></a>Note

> [!WARNING]
> Il campo `TlsStream.m_Worker` è privato e non è destinato a essere usato direttamente nel codice.
>
> Microsoft non supporta l'utilizzo di questo campo in un'applicazione di produzione in qualsiasi circostanza.
