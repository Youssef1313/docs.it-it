# <a name="windows-communication-foundation-bindings"></a>Associazioni di Windows Communication Foundation
Le associazioni specificano il modo in cui un endpoint del servizio Windows Communication Foundation (WCF) comunica con altri endpoint. In parole semplici, un'associazione deve specificare il trasporto (ad esempio HTTP o TCP) da utilizzare. Le associazioni consentono inoltre di impostare altre caratteristiche, ad esempio la protezione e il supporto delle transazioni.
