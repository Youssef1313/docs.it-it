# <a name="wcf-system-requirements"></a>Requisiti di sistema per WCF

Windows Communication Foundation (WCF) è un'infrastruttura di comunicazione utilizzata per creare applicazioni distribuite. Di seguito sono elencati i requisiti che consentono l'esecuzione di WCF.

## <a name="system-requirements"></a>Requisiti di sistema

WCF viene installato per impostazione predefinita in [!INCLUDE[wv](../../../includes/wv-md.md)].

È anche possibile installare WCF in [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], Windows Server 2003 R2 o [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].

> [!NOTE]
> La funzionalità di Accodamento messaggi (MSMQ) di WCF è supportata solo su [!INCLUDE[wv](../../../includes/wv-md.md)], Windows Server 2003 R2, [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] e Windows XP Professional.
