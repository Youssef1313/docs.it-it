# <a name="wmi-class-reference"></a>Riferimento alle classi WMI
In questa sezione sono elencate tutte le classi WMI esposte dal provider WMI Windows Communication Foundation (WCF).  
  
## <a name="accessing-wmi-instances"></a>Accesso alle istanze WMI  
 Di tutte le classi elencate nel riferimento agli oggetti WMI non è possibile creare direttamente un'istanza, ad eccezione di Service, AppDomain, Contract, ServiceAppDomain, ServiceToEndpointAssociation e Endpoint. Per accedere alle altre istanze, è possibile accedere alle proprietà delle classi del livello superiore precedentemente menzionate. Ad esempio, è possibile accedere all'istanza di TransportBindingElement dall'istanza di endpoint-> Binding-> BindingElement.
