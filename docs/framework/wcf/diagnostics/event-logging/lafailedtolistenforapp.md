# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
Id: 160  
  
 Gravità: Errore  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificato un errore nel servizio di attivazione specificato del protocollo specificato, durante il tentativo di ascolto del sito specificato, pertanto il protocollo è temporaneamente disabilitato per il sito. L'evento elenca l'URL, lo stato, l'eccezione, il nome del processo e l'ID del processo.
