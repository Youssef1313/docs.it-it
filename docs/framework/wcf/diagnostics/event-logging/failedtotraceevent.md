# <a name="failedtotraceevent"></a>FailedToTraceEvent
Id: 104  
  
 Gravità: Errore  
  
 Categoria: Traccia  
  
## <a name="description"></a>Descrizione  
 Uno o più eventi non sono tracciati. Vengono indicati la stringa originale dell'evento, l'ID e il nome del processo.
