# <a name="complustlbimporterror"></a>ComPlusTLBImportError
Id: 132  
  
 Gravità: Errore  
  
 Categoria: ServiceModel  
  
## <a name="description"></a>DESCRIZIONE  
 Questo evento indica che si è verificato un errore durante l'importazione della libreria dei tipi. Nell'evento sono indicati l'IDD, l'ID della libreria dei tipi, l'eccezione, il nome e l'ID del processo.
