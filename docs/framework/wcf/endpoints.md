# <a name="windows-communication-foundation-endpoints"></a>Endpoint di Windows Communication Foundation
Tutte le comunicazioni con un servizio Windows Communication Foundation (WCF) avvengono tramite gli *endpoint* del servizio. Gli endpoint forniscono ai client l'accesso alla funzionalità offerta da un servizio WCF.  
  
 Per una panoramica su come creare un endpoint, vedere [Cenni preliminari sulla creazione di endpoint](endpoint-creation-overview.md). Ogni endpoint contiene:  
  
- Un indirizzo che indica dove si trova l'endpoint.  
  
- Un'associazione che specifica in che modo un client può comunicare con l'endpoint.  
  
- Un contratto che identifica i metodi disponibili.  
  
 Per istruzioni su come specificare queste singole parti di un endpoint, vedere:  
  
- [Specifica di un indirizzo dell'endpoint](specifying-an-endpoint-address.md)  
  
- [Uso di associazioni per configurare servizi e client](using-bindings-to-configure-services-and-clients.md)  
  
- [Progettazione e implementazione di servizi](designing-and-implementing-services.md)
