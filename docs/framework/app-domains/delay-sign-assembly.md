<span data-ttu-id="6774c-103">È possibile che in un'organizzazione venga usata una coppia di chiavi sottoposta a una rigorosa sicurezza e quindi non accessibile giornalmente agli sviluppatori.</span><span class="sxs-lookup"><span data-stu-id="6774c-103">An organization can have a closely guarded key pair that developers do not have access to on a daily basis.</span></span> <span data-ttu-id="6774c-104">La chiave pubblica è spesso disponibile, ma l'accesso alla chiave privata è consentito solo ad alcune persone.</span><span class="sxs-lookup"><span data-stu-id="6774c-104">The public key is often available, but access to the private key is restricted to only a few individuals.</span></span> <span data-ttu-id="6774c-105">Quando si sviluppano assembly con nome sicuro, in ogni assembly che fa riferimento all'assembly con nome sicuro di destinazione è contenuto un token della chiave pubblica usata per assegnare un nome sicuro all'assembly di destinazione.</span><span class="sxs-lookup"><span data-stu-id="6774c-105">When developing assemblies with strong names, each assembly that references the strong-named target assembly contains the token of the public key used to give the target assembly a strong name.</span></span> <span data-ttu-id="6774c-106">È quindi necessario che la chiave pubblica risulti disponibile durante il processo di sviluppo.</span><span class="sxs-lookup"><span data-stu-id="6774c-106">This requires that the public key be available during the development process.</span></span>  
  
 <span data-ttu-id="6774c-107">È possibile usare il ritardo della firma o la firma parziale in fase di compilazione per riservare nel file eseguibile trasportabile (PE, Portable Executable) lo spazio per la firma con nome sicuro, posticipando la firma effettiva a una fase successiva, solitamente prima della consegna dell'assembly.</span><span class="sxs-lookup"><span data-stu-id="6774c-107">You can use delayed or partial signing at build time to reserve space in the portable executable (PE) file for the strong name signature, but defer the actual signing until some later stage (typically just before shipping the assembly).</span></span>  
  
 <span data-ttu-id="6774c-108">I passaggi seguenti illustrano il processo che consente di ritardare la firma di un assembly:</span><span class="sxs-lookup"><span data-stu-id="6774c-108">The following steps outline the process to delay sign an assembly:</span></span>

3. <span data-ttu-id="6774c-115">Il compilatore inserisce la chiave pubblica nel manifesto dell'assembly e riserva nel file PE lo spazio necessario per la firma completa con nome sicuro.</span><span class="sxs-lookup"><span data-stu-id="6774c-115">The compiler inserts the public key into the assembly manifest and reserves space in the PE file for the full strong name signature.</span></span> <span data-ttu-id="6774c-116">La chiave pubblica reale deve essere memorizzata in fase di compilazione dell'assembly, in modo che gli altri assembly contenenti riferimenti a questo assembly siano in grado di ottenere la chiave da memorizzare nei relativi riferimenti di assembly.</span><span class="sxs-lookup"><span data-stu-id="6774c-116">The real public key must be stored while the assembly is built so that other assemblies that reference this assembly can obtain the key to store in their own assembly reference.</span></span>  
  
4. <span data-ttu-id="6774c-117">Poiché l'assembly non dispone di una firma con nome sicuro valida, è necessario disattivare la verifica della firma di tale assembly.</span><span class="sxs-lookup"><span data-stu-id="6774c-117">Because the assembly does not have a valid strong name signature, the verification of that signature must be turned off.</span></span> <span data-ttu-id="6774c-118">A tale scopo, usare l'opzione **-Vr** dello strumento Nome sicuro.</span><span class="sxs-lookup"><span data-stu-id="6774c-118">You can do this by using the **–Vr** option with the Strong Name tool.</span></span>


    ```  
    sn –Vk myRegFile.reg myAssembly.dll  
    ```  
  
     <span data-ttu-id="6774c-123">Con l'opzione **-Vr** o **-Vk** è possibile includere un file con estensione snk per la firma con una chiave di test.</span><span class="sxs-lookup"><span data-stu-id="6774c-123">With either the **–Vr** or **–Vk** option, you can optionally include an .snk file for test key signing.</span></span>
