# <a name="breaking-changes-in-windows-forms-net-core-to-net-core"></a>Modifiche di rilievo in Windows Forms (.NET Core a .NET Core)

È stato aggiunto il supporto Windows Forms a .NET Core nella versione 3,0. Questo articolo elenca le modifiche di rilievo per Windows Forms dalla versione di .NET Core in cui sono state introdotte. Se si sta aggiornando un'app Windows Forms da una versione precedente di .NET Core (3,0 o versione successiva), questo articolo è applicabile all'utente. Se si esegue la migrazione di un'app Windows Forms da .NET Framework a .NET Core, vedere [modifiche di rilievo per Windows Forms (.NET Framework a .NET Core)](../porting/winforms-breaking-changes.md).
