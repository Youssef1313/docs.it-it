# <a name="get-started"></a>Introduzione

Per iniziare a usare .NET, sono disponibili vari metodi. Poiché .NET è una piattaforma di grandi dimensioni, sono disponibili più articoli in questa documentazione che mostrano com'è possibile iniziare a usare .NET. Ogni articolo tratta un aspetto diverso.

## <a name="get-started-using-net-languages"></a>Introduzione all'uso dei linguaggi .NET

* Gli articoli di [introduzione a C#](../csharp/getting-started/index.md) e le [esercitazioni su C#](../csharp/tutorials/index.md) forniscono una serie di metodi per iniziare a usare C#.

* Le esercitazioni [introduttive con F#](../fsharp/get-started/index.md) offrono tre metodi principali per l'uso di F#: con Visual Studio, Visual Studio Code o gli strumenti da riga di comando.

* Gli articoli di [introduzione a Visual Basic](../visual-basic/getting-started/index.md) forniscono guide per l'uso di Visual Basic in Visual Studio.

## <a name="get-started-using-net-core"></a>Attività iniziali con .NET Core

* [Introduzione a .NET Core](../core/get-started.md) fornisce una panoramica di articoli che illustrano come iniziare a usare .NET Core su diversi sistemi operativi e con diversi strumenti.

* Le [esercitazioni di .NET Core](../core/tutorials/index.md) descrivono vari metodi per iniziare a usare .NET Core con il sistema operativo e gli strumenti preferiti.

## <a name="get-started-using-net-core-on-docker"></a>Introduzione all'uso di .NET Core in Docker

L'[Introduzione a .NET e Docker](../core/docker/introduction.md) illustra come usare .NET Core nei contenitori Windows Docker.
